# Article L511-1

Les prestations familiales comprennent :    1°) la prestation d'accueil du jeune enfant ;   2°) les allocations familiales ;    3°) le complément familial ;    4°) l'allocation de logement ;    5°) l'allocation d'éducation de l'enfant handicapé ;   6°) l'allocation de soutien familial ;    7°) l'allocation de rentrée scolaire ;    8°) l'allocation de parent isolé ;    9°) l'allocation de présence parentale.
