# Article L114-16-2

<p>Les fraudes en matière sociale mentionnées à l'article L. 114-16-1 sont celles définies par : </p><p>― les <a href='/code-penal/partie-legislative/livre-iii-des-crimes-et-delits-contre-les-biens/titre-ier-des-appropriations-frauduleuses/chapitre-iii-de-lescroquerie-et-des-infractions-voisines/section-1-de-lescroquerie/313-1.md'>articles 313-1, 441-1, 441-6 et 441-7 du code pénal </a>lorsqu'elles portent un préjudice aux organismes de protection sociale ; </p><p>― les articles <a href='/code-de-la-securite-sociale/partie-legislative/livre-ii-organisation-du-regime-general-action-de-prevention-action-sanitaire-et-sociale-des-caisses/titre-vii-dispositions-diverses/chapitre-2-sanctions/l272-1.md'>L. 272-1</a>, <a href='/code-de-la-securite-sociale/partie-legislative/livre-iii-dispositions-relatives-aux-assurances-sociales-et-a-diverses-categories-de-personnes-rattachees-au-regime-general/titre-vii-dispositions-diverses/chapitre-7-penalites/l377-5.md'>L. 377-5</a>, <a href='/code-de-la-securite-sociale/partie-legislative/livre-v-prestations-familiales-et-prestations-assimilees/titre-viii-dispositions-relatives-au-recouvrement-des-creances-alimentaires-dispositions-diverses-dispositions-dapplication/chapitre-3-dispositions-diverses/l583-3.md'>L. 583-3 </a>et <a href='/code-de-la-securite-sociale/partie-legislative/livre-8-allocations-aux-personnes-agees-allocation-aux-adultes-handicapes-allocation-de-logement-sociale-aides-a-lemploi-pour-la-garde-des-jeunes-enfants-aides-aux-collectivites-et-organismes-logeant-a-titre-temporaire-des-personnes-defavorisee/titre-iii-allocation-de-logement-des-personnes-agees-des-infirmes-des-jeunes-salaries-et-de-certaines-categories-de-demandeurs-demploi/chapitre-1er-conditions-generales-dattribution/section-1-dispositions-communes/l831-7.md'>L. 831-7 </a>du présent code ; </p><p>― l'article <a href='/code-de-laction-sociale-et-des-familles/partie-legislative/livre-iii-action-sociale-et-medico-sociale-mise-en-oeuvre-par-des-etablissements-et-des-services/titre-v-contentieux-de-la-tarification-sanitaire-et-sociale/chapitre-unique/l351-1.md'>L. 351-1</a> du code de l'action sociale et des familles ; </p><p>― les articles <a href='/code-de-la-construction-et-de-lhabitation/partie-legislative/livre-iii-aides-diverses-a-la-construction-dhabitations-et-a-lamelioration-de-lhabitat-aide-personnalisee-au-logement/titre-v-aide-personnalisee-au-logement/chapitre-ier-dispositions-generales/l351-12.md'>L. 351-12 </a>et <a href='/code-de-la-construction-et-de-lhabitation/partie-legislative/livre-vi-mesures-tendant-a-remedier-a-des-difficultes-exceptionnelles-de-logement/titre-v-sanctions-et-dispositions-diverses/chapitre-unique/l651-1.md'>L. 651-1 </a>du code de la construction et de l'habitation ; </p><p>― les <a href='/code-du-travail/partie-legislative/cinquieme-partie-lemploi/livre-ier-les-dispositifs-en-faveur-de-lemploi/titre-ii-aides-au-maintien-et-a-la-sauvegarde-de-lemploi/chapitre-iv-dispositions-penales/l5124-1.md'>articles L. 5124-1, L. 5413-1, L. 5429-1, L. 5429-3 et L. 5522-28 du code du travail </a>; </p><p>― l'<a href='/affichTexteArticle.do?cidTexte=JORFTEXT000000317514&idArticle=LEGIARTI000006755889&dateTexte=&categorieLien=cid'>article 22 de la loi n° 68-690 du 31 juillet 1968 </a>portant diverses dispositions d'ordre économique et financier.</p>
