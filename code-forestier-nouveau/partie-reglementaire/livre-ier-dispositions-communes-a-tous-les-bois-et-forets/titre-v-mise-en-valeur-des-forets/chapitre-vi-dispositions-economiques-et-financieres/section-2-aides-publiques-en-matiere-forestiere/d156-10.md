# Article D156-10

<p><br/>Les barèmes prévus à l'article D. 156-9 sont établis par le préfet de région conformément aux principes suivants : <br/>1° Catégories de travaux autorisés sur barème : <br/>a) Travaux de nettoyage ; <br/>b) Travaux de reconstitution ; <br/>c) Travaux d'entretien ; <br/>2° Nombre de barèmes autorisés selon le type de travaux : <br/>a) Travaux de nettoyage : 3. <br/>Chaque niveau de barème peut être décliné en trois tranches de surfaces maximum ; <br/>b) Travaux de reconstitution par régénération artificielle : 5, dont 2 pour les techniques de semis et 3 pour les techniques de plantation. <br/>Quatre options sont possibles : <br/>― une option assistance technique ; <br/>― une option étude écologique ou paysagère ; <br/>― une option de protection contre le gibier ; <br/>― une option technique laissée à l'appréciation du préfet lorsque celle-ci est justifiée par l'adaptation aux spécificités locales ; <br/>c) Travaux de régénération naturelle : 3. Chaque niveau de barème peut être décliné en trois tranches de surfaces maximum. <br/>Cinq options sont possibles : <br/>― une option assistance technique ; <br/>― une option étude écologique ou paysagère ; <br/>― une option de protection contre le gibier ; <br/>― une option spécifique à la régénération naturelle sur la base d'un enrichissement par plantation d'un nombre de feuillus précieux à l'hectare, protégés individuellement contre le gibier ; <br/>― une option technique laissée à l'appréciation du préfet lorsque celle-ci est justifiée par l'adaptation aux spécificités locales ; <br/>d) Travaux d'entretien : 1. <br/>Une seule option est possible : assistance technique.</p>
