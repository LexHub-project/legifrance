# Article R221-4

<p>Les tribunaux administratifs désignés ci-après sont composés de plusieurs chambres, dont le nombre est fixé comme suit : <br/><br/> Amiens : quatre chambres ; <br/><br/> Bastia : deux chambres ; <br/><br/> Besançon : deux chambres ; <br/><br/> Bordeaux : cinq chambres ; <br/><br/> Caen : deux chambres ; <br/><br/> Cergy-Pontoise : dix chambres ; <br/><br/> Châlons-en-Champagne : trois chambres ; <br/><br/> Clermont-Ferrand : deux chambres ; <br/><br/> Dijon : trois chambres ; <br/><br/> Grenoble : cinq chambres ; <br/><br/> Lille : six chambres ; <br/><br/> Limoges : deux chambres ; <br/><br/> Lyon : sept chambres ; <br/><br/> Marseille : huit chambres ; <br/><br/> Melun : sept chambres ; <br/><br/> Montpellier : sept chambres ; <br/><br/> Nancy : deux chambres ; <br/><br/> Nantes : cinq chambres ; <br/><br/> Nice : sept chambres ; <br/><br/> Nîmes : trois chambres ; <br/><br/> Orléans : cinq chambres ; <br/><br/> Pau : deux chambres ; <br/><br/> Poitiers : trois chambres ; <br/><br/> Rennes : cinq chambres ; <br/><br/> Rouen : trois chambres ; <br/><br/> Strasbourg : cinq chambres ; <br/><br/> Toulon : trois chambres ; <br/><br/> Toulouse : cinq chambres ; <br/><br/> Versailles : dix chambres ; <br/><br/> Saint-Denis : deux chambres.</p>
