# Article R41-11

<p>En application de l'article <a href='/code-de-procedure-penale/partie-legislative/livre-ii-des-juridictions-de-jugement/titre-iii-du-jugement-des-contraventions/chapitre-ier-de-la-competence-du-tribunal-de-police-et-de-la-juridiction-de-proximite/523.md'>523</a>, le tribunal de police ne peut être constitué par un magistrat exerçant à titre temporaire pour le jugement des contraventions suivantes : </p><p>1° Diffamation non publique prévue par <a href='/code-penal/partie-reglementaire-decrets-en-conseil-detat/livre-vi-des-contraventions/titre-ii-des-contraventions-contre-les-personnes/chapitre-ier-des-contraventions-de-la-1re-classe-contre-les-personnes/section-1-de-la-diffamation-et-de-linjure-non-publiques/r621-1.md'>l'article R. 621-1 du code pénal </a>; </p><p>2° Injure non publique prévue par <a href='/code-penal/partie-reglementaire-decrets-en-conseil-detat/livre-vi-des-contraventions/titre-ii-des-contraventions-contre-les-personnes/chapitre-ier-des-contraventions-de-la-1re-classe-contre-les-personnes/section-1-de-la-diffamation-et-de-linjure-non-publiques/r621-2.md'>l'article R. 621-2 du code pénal </a>; </p><p>3° Diffamation non publique présentant un caractère raciste ou discriminatoire prévue par <a href='/code-penal/partie-reglementaire-decrets-en-conseil-detat/livre-vi-des-contraventions/titre-ii-des-contraventions-contre-les-personnes/chapitre-iv-des-contraventions-de-la-4e-classe-contre-les-personnes/section-3-de-la-diffamation-et-de-linjure-non-publiques-presentant-un-caractere-raciste-ou-discriminatoire/r624-3.md'>l'article R. 624-3 du code pénal </a>; </p><p>4° Injure non publique présentant un caractère raciste ou discriminatoire prévue par <a href='/code-penal/partie-reglementaire-decrets-en-conseil-detat/livre-vi-des-contraventions/titre-ii-des-contraventions-contre-les-personnes/chapitre-iv-des-contraventions-de-la-4e-classe-contre-les-personnes/section-3-de-la-diffamation-et-de-linjure-non-publiques-presentant-un-caractere-raciste-ou-discriminatoire/r624-4.md'>l'article R. 624-4 du code pénal</a>.</p>
