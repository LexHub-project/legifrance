# Article A43-12

<p>Le tarif de l'enquête sociale mentionnée aux articles <a href='/code-de-procedure-civile/livre-iii-dispositions-particulieres-a-certaines-matieres/titre-ier-les-personnes/chapitre-v-la-procedure-en-matiere-familiale/section-i-dispositions-generales/1072.md'>1072 </a>et <a href='/code-de-procedure-civile/livre-iii-dispositions-particulieres-a-certaines-matieres/titre-ier-les-personnes/chapitre-x-la-tutelle-des-mineurs/section-i-le-juge-des-tutelles/sous-section-5-la-sauvegarde-de-justice/1248.md'>1248</a> du code de procédure civile est fixé à 500 euros. </p><p>Le montant de l'indemnité de carence est fixé à 30 euros.</p><br/><br/><i>NOTA:<p>Les dispositions contenues dans l'article 1248 du code de procédure civile ont été intégrées dans l'article 1221 du même code par l'article 1er du décret n° 2008-1276 du 5 décembre 2008.</p></i>
