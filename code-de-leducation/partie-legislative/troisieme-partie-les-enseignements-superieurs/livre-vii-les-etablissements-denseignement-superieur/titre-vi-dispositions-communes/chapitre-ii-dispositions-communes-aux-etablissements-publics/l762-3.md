# Article L762-3

<p>Dans les conditions prévues à <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071190&idArticle=LEGIARTI000006524223&dateTexte=&categorieLien=cid'>l'article L. 321-6 du code de la recherche</a>, les établissements publics d'enseignement supérieur peuvent confier par convention à des personnes morales de droit privé les activités mentionnées à l'article <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071190&idArticle=LEGIARTI000006524221&dateTexte=&categorieLien=cid'>L. 321-5</a> du même code.</p>
