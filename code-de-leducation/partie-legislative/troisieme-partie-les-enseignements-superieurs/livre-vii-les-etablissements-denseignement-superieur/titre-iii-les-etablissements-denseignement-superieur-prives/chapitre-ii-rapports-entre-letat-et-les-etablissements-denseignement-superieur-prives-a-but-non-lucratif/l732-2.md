# Article L732-2

<div align='left'>L'établissement ayant obtenu la qualification d'établissement d'enseignement supérieur privé d'intérêt général dans les conditions prévues à l'article L. 732-1 du présent code conclut avec l'Etat un contrat pluriannuel d'établissement. Ce contrat définit les conditions dans lesquelles l'établissement exerce les missions du service public de l'enseignement supérieur, dans le cadre d'une gestion désintéressée au sens du <a href='/code-general-des-impots-cgi/livre-premier-assiette-et-liquidation-de-limpot/premiere-partie-impots-detat/titre-ii-taxes-sur-le-chiffre-daffaires-et-taxes-assimilees/chapitre-premier-taxe-sur-la-valeur-ajoutee/section-i-champ-dapplication/iii-operations-exonerees/261.md'>d du 1° du 7 de l'article 261 du code général des impôts</a>.</div>
