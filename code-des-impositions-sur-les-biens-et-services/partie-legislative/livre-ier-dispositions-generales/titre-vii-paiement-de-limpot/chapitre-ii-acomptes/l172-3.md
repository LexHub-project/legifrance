# Article L172-3

<p><br/>Des décrets déterminent, pour chaque imposition :<br/>1° Les catégories de redevables concernés ;<br/>2° Les échéances des acomptes, qui ne peuvent être antérieures au premier jour de l'année civile qui précède la déclaration ;<br/>3° Le nombre des acomptes, qui ne peut excéder douze ;<br/>4° Les règles de détermination des acomptes et, le cas échéant, les conditions dans lesquelles les redevables peuvent y déroger.</p>
