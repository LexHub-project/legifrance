# Article R2461-3

<p>Pour l'application de l'article <a href='/code-de-la-defense/partie-reglementaire/partie-2-regimes-juridiques-de-defense/livre-ier-regimes-dapplication-exceptionnelle/titre-ier-guerre/chapitre-ii-dispositions-applicables-aux-communes/r2112-1.md'>R. 2112-1 </a>en Nouvelle-Calédonie, les mots : aux <a href='/code-general-des-collectivites-territoriales/partie-reglementaire/deuxieme-partie-la-commune/livre-ier-organisation-de-la-commune/titre-ii-organes-de-la-commune/chapitre-iv-dispositions-applicables-en-periode-de-mobilisation-generale-et-en-temps-de-guerre/r2124-2.md'>articles R. 2124-2 à R. 2124-5 </a>du code général des collectivités territoriales sont remplacés par les mots : aux articles <a href='/code-des-communes-de-la-nouvelle-caledonie/partie-reglementaire/livre-ier-organisation-communale/titre-ii-organes-de-la-commune/chapitre-iv-dispositions-applicables-en-periodes-de-mobilisation-et-de-temps-de-guerre/section-1-dispositions-applicables-aux-conseils-et-aux-conseillers-municipaux/r124-1.md'>R. 124-1 </a>et <a href='/code-des-communes-de-la-nouvelle-caledonie/partie-reglementaire/livre-ier-organisation-communale/titre-ii-organes-de-la-commune/chapitre-iv-dispositions-applicables-en-periodes-de-mobilisation-et-de-temps-de-guerre/section-1-dispositions-applicables-aux-conseils-et-aux-conseillers-municipaux/r124-3.md'>R. 124-3 à R. 124-6</a> du code des communes de Nouvelle-Calédonie.</p>
