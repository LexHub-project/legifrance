# Article L6242-5

Les personnes morales déclarées pénalement responsables, dans les conditions prévues par <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>l'article 121-2 </a>du code pénal, de l'infraction définie à <a href='/code-de-la-sante-publique/partie-legislative/sixieme-partie-etablissements-et-services-de-sante/livre-ii-laboratoires-danalyses-de-biologie-medicale/titre-iv-sanctions/chapitre-ii-sanctions-penales/l6242-2.md'>l'article L. 6242-2 </a>encourent, outre l'amende suivant les modalités prévues par <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>l'article 131-38</a>, les peines prévues par les articles 2° à 9° de <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>l'article 131-39</a>.<br/><br/>L'interdiction mentionnée au 2° de l'article 131-39 du même code porte sur une ou plusieurs professions régies par le présent code ou toute autre activité professionnelle ou sociale à l'occasion de l'exercice de laquelle l'infraction a été commise.
