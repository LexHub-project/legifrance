# Article R5122-1

<p>Les éléments contenus dans la publicité pour un médicament doivent être conformes aux renseignements figurant dans le résumé des caractéristiques du produit mentionné à <a href='/code-de-la-sante-publique/partie-reglementaire/cinquieme-partie-produits-de-sante/livre-ier-produits-pharmaceutiques/titre-ii-medicaments-a-usage-humain/chapitre-ier-dispositions-generales/section-5-autorisation-de-mise-sur-le-marche/sous-section-1-demande-dautorisation/r5121-21.md'>l'article R. 5121-21</a>. Toute publicité pour un médicament traditionnel à base de plantes comporte, outre les mentions prévues aux <a href='/code-de-la-sante-publique/partie-reglementaire/cinquieme-partie-produits-de-sante/livre-ier-produits-pharmaceutiques/titre-ii-medicaments-a-usage-humain/chapitre-ii-publicite/section-2-publicite-aupres-du-public/r5122-3.md'>articles R. 5122-3 </a>et <a href='/code-de-la-sante-publique/partie-reglementaire/cinquieme-partie-produits-de-sante/livre-ier-produits-pharmaceutiques/titre-ii-medicaments-a-usage-humain/chapitre-ii-publicite/section-3-publicite-a-destination-des-professions-de-sante/r5122-8.md'>R. 5122-8</a>, la mention : " Médicament traditionnel à base de plantes à utiliser ", suivie des indications à spécifier, elles-mêmes suivies de la mention : " sur la base exclusive de l'ancienneté de l'usage ". Seules les mentions prévues à l'article <a href='/code-de-la-sante-publique/partie-reglementaire/cinquieme-partie-produits-de-sante/livre-ier-produits-pharmaceutiques/titre-ii-medicaments-a-usage-humain/chapitre-ier-dispositions-generales/section-11-etiquetage/sous-section-4-medicaments-soumis-a-enregistrement/r5121-146.md'>R. 5121-146 </a>peuvent être utilisées dans la publicité pour les médicaments homéopathiques mentionnés à <a href='/code-de-la-sante-publique/partie-legislative/cinquieme-partie-produits-de-sante/livre-ier-produits-pharmaceutiques/titre-ii-medicaments-a-usage-humain/chapitre-ier-dispositions-generales/l5121-13.md'>l'article L. 5121-13</a>.</p>
