# Article D4031-45-1

<p>Le taux annuel de la contribution est fixé, par profession, comme suit : </p><p> 1° Pour les médecins : 0,5 % ; </p><p> 2° Pour les chirurgiens-dentistes, les pharmaciens et les biologistes responsables : 0,3 % ; </p><p> 3° Pour les infirmiers, les masseurs-kinésithérapeutes, les pédicures-podologues, les sages-femmes, les orthophonistes et les orthoptistes : 0,1 %. </p>
