# Article R4031-6

<p>Le nombre total des membres de l'assemblée de chaque union régionale est fixé comme suit : <br/><br/>I. ― Pour l'union régionale regroupant les médecins : <br/><br/>1° Dix membres dans les régions où le nombre de médecins exerçant à titre libéral dans le cadre du régime conventionnel est inférieur ou égal à 500 ; <br/><br/>2° Trente membres dans les régions où le nombre de médecins exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 501 et 3 000 ; <br/><br/>3° Quarante membres dans les régions où le nombre de médecins exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 3 001 et 5 000 ; <br/><br/>4° Soixante membres dans les régions où le nombre de médecins exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 5 001 et 10 000 ; <br/><br/>5° Quatre-vingts membres dans les régions où le nombre de médecins exerçant à titre libéral dans le cadre du régime conventionnel est supérieur à 10 000. <br/><br/>II. ― Pour chaque union régionale regroupant les professionnels élisant leurs représentants : <br/><br/>1° Trois membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est inférieur ou égal à 200 ; <br/><br/>2° Six membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 201 et 700 ; <br/><br/>3° Neuf membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 701 et 1 500 ; <br/><br/>4° Douze membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 1 501 et 2 500 ; <br/><br/>5° Quinze membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 2 501 et 3 500 ; </p><p>6° Dix-huit membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 3 501 et 5 000 ; <br/><br/>7° Vingt-quatre membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est supérieur à 5 000. </p><p>III. ― Pour chaque union régionale regroupant les professionnels désignant leurs représentants : <br/><br/> 1° Trois membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est inférieur ou égal à 100 ; <br/><br/> 2° Six membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 101 et 300 ; <br/><br/> 3° Neuf membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est compris entre 301 et 500 ; <br/><br/> 4° Douze membres dans les régions où le nombre de ces professionnels de santé exerçant à titre libéral dans le cadre du régime conventionnel est supérieur à 500. <br/><br/> IV. ― Pour l'application du présent article, le nombre de professionnels de santé pris en compte est celui au premier jour du quatrième mois précédant le renouvellement de l'assemblée sortante. Ce nombre est communiqué au président de l'union régionale des professionnels de santé par les caisses primaires d'assurance maladie de la région. <br/><br/> Lorsqu'un professionnel de santé exerce dans plusieurs régions, il est pris en compte dans la région où il exerce à titre principal. </p>
