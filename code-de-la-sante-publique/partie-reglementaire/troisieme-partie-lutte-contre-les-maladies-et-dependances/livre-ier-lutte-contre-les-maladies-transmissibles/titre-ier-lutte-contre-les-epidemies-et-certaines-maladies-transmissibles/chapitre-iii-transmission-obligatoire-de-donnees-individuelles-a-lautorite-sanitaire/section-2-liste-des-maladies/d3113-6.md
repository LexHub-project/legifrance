# Article D3113-6

<p>La liste des maladies qui relèvent de la procédure de signalement prévue à l'article <a href='/code-de-la-sante-publique/partie-reglementaire/troisieme-partie-lutte-contre-les-maladies-et-dependances/livre-ier-lutte-contre-les-maladies-transmissibles/titre-ier-lutte-contre-les-epidemies-et-certaines-maladies-transmissibles/chapitre-iii-transmission-obligatoire-de-donnees-individuelles-a-lautorite-sanitaire/section-1-procedures-de-transmission/r3113-4.md'>R. 3113-4</a> est la suivante : </p><p>1° Maladies infectieuses : </p><p>a) Botulisme ; </p><p>b) Brucellose ; </p><p>c) Charbon ; </p><p>c) 1. Chikungunya ; </p><p>d) Choléra ; </p><p>d) 1. Dengue ; </p><p>e) Diphtérie ; </p><p>f) Fièvres hémorragiques africaines ; </p><p>g) Fièvre jaune ; </p><p>h) Fièvre typhoïde et fièvres paratyphoïdes ; </p><p>i) Hépatite A aiguë ; </p><p>j) Infection invasive à méningocoque ; </p><p>k) Légionellose ; </p><p>l) Listériose ; </p><p>m) Orthopoxviroses, dont la variole ; </p><p>n) Paludisme autochtone ; </p><p>o) Paludisme d'importation dans les départements d'outre-mer ; </p><p>p) Peste ; </p><p>q) Poliomyélite ; </p><p>r) Rage ; </p><p>s) Rougeole ; </p><p>s) 1. Schistosomiase (Bilharziose) urogénitale autochtone ;</p><p>t) Suspicion de maladie de Creutzfeldt-Jakob et autres encéphalopathies subaiguës spongiformes transmissibles humaines ; </p><p>u) Toxi-infections alimentaires collectives ; </p><p>v) Tuberculose ; </p><p>w) Tularémie ; </p><p>x) Typhus exanthématique ; </p><p>y) Zika ; </p><p>2° Autre maladie :</p><p>- saturnisme chez les enfants mineurs.</p><p>Une maladie peut être retirée de cette liste, par arrêté du ministre chargé de la santé, pour tout département ou collectivité dont la situation épidémique le justifie.</p>
