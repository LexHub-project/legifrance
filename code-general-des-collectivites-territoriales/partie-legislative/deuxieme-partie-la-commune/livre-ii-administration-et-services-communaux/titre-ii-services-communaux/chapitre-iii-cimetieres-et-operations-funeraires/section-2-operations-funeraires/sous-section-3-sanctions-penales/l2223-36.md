# Article L2223-36

Les personnes morales peuvent être déclarées responsables pénalement, dans les conditions prévues par l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md' title='Code pénal - art. 121-2 (V)'>121-2 </a>du code pénal, des infractions définies à l'article <a href='/code-general-des-collectivites-territoriales/partie-legislative/deuxieme-partie-la-commune/livre-ii-administration-et-services-communaux/titre-ii-services-communaux/chapitre-iii-cimetieres-et-operations-funeraires/section-2-operations-funeraires/sous-section-3-sanctions-penales/l2223-35.md' title='Code général des collectivités territoriales - art. L2223-35 (V)'>L. 2223-35</a>. Les peines encourues par les personnes morales sont : 1° L'amende, suivant les modalités prévues par l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md' title='Code pénal - art. 131-38 (V)'>131-38 </a>du code pénal ; 2° Les peines mentionnées aux 2°, 3°, 4°, 5°, 6°, 7°, 8° et 9° de l'article <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006070719&idArticle=LEGIARTI000006417335&dateTexte=&categorieLien=cid' title='Code pénal - art. 131-39 (V)'>131-39</a> du code pénal ; l'interdiction mentionnée au 2° de l'article 131-39 du même code porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.
