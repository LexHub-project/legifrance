# Article R221-52

<p>Dans le cas prévu à <a href='/code-civil/livre-ier-des-personnes/titre-ier-bis-de-la-nationalite-francaise/chapitre-vi-du-contentieux-de-la-nationalite/section-3-des-certificats-de-nationalite-francaise/31-1.md'>l'article 31-1</a> du code civil, la demande est portée devant :</p><p>1° Le tribunal dans le ressort duquel est situé le domicile du demandeur, si celui-ci réside en France ;</p><p>2° Le tribunal dans le ressort duquel est situé le lieu de naissance du demandeur, si celui-ci est né en France et réside à l'étranger ;</p><p>3° Le tribunal d'instance de Paris, si le demandeur est né et réside à l'étranger.</p>
