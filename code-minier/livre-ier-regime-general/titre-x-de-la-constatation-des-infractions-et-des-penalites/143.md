# Article 143

<p>Les personnes morales déclarées responsables pénalement, dans les conditions prévues par l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>article 121-2 du code pénal</a>, des infractions définies aux <a href='/code-minier/livre-ier-regime-general/titre-x-de-la-constatation-des-infractions-et-des-penalites/141.md'>articles 141, 141-1 et 142 </a>du présent code encourent, outre l'amende suivant les modalités prévues par l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>article 131-38 du code pénal</a>, les peines prévues par les 2° à 6°, 8° et 9° de <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>l'article 131-39</a> du même code.</p><p>L'interdiction mentionnée au 2° de l'article 131-39 du même code porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.</p>
