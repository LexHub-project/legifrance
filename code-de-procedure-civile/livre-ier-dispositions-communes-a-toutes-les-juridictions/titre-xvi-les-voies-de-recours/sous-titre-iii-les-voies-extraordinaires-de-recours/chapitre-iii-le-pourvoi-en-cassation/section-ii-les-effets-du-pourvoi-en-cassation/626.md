# Article 626

<p>En cas de cassation suivie d'un renvoi de l'affaire à une juridiction, celle-ci est désignée et statue, le cas échéant, conformément à <a href='/code-de-lorganisation-judiciaire/partie-legislative/livre-iv-la-cour-de-cassation/titre-iii-fonctionnement/chapitre-ier-les-chambres-de-la-cour/section-1-dispositions-generales/l431-4.md'>l'article L. 431-4</a> du code de l'organisation judiciaire.</p>
