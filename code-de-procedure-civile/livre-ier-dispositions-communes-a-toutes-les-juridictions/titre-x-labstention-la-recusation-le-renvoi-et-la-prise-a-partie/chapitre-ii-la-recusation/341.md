# Article 341

<p>Sauf disposition particulière, la récusation d'un juge est admise pour les causes prévues par <a href='/code-de-lorganisation-judiciaire/partie-legislative/livre-ier-dispositions-communes-aux-juridictions-judiciaires/titre-ier-principes-generaux/chapitre-unique/l111-6.md'>l'article L. 111-6</a> du code de l'organisation judiciaire.</p>
