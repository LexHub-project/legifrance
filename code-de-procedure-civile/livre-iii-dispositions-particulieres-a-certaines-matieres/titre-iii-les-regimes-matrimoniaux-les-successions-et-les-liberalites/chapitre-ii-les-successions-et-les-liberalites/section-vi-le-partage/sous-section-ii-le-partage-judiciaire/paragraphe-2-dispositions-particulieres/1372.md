# Article 1372

Si un acte de partage amiable est établi, en application des <a href='/code-civil/livre-iii-des-differentes-manieres-dont-on-acquiert-la-propriete/titre-ier-des-successions/chapitre-viii-du-partage/section-1-des-operations-de-partage/sous-section-3-du-partage-judiciaire/842.md' title='Code civil - art. 842 (M)'>dispositions de l'article 842 du code civil</a>, le notaire en informe le juge qui constate la clôture de la procédure.
