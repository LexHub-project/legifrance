# Article 1260-7

<p>En application des <a href='/code-civil/livre-ier-des-personnes/titre-xi-de-la-majorite-et-des-majeurs-proteges-par-la-loi/chapitre-ii-des-mesures-de-protection-juridique-des-majeurs/section-6-de-lhabilitation-familiale/494-4.md'>dispositions du deuxième alinéa de l'article 494-4 du code civil</a>, le juge constate l'adhésion ou l'absence d'opposition légitime des personnes visées au <a href='/code-civil/livre-ier-des-personnes/titre-xi-de-la-majorite-et-des-majeurs-proteges-par-la-loi/chapitre-ii-des-mesures-de-protection-juridique-des-majeurs/section-6-de-lhabilitation-familiale/494-1.md'>deuxième alinéa de l'article 494-1 du code civil</a> après les avoir entendus ou par écrit. <br/></p>
