# Article 1217

<p>Hors les cas prévus aux <a href='/code-civil/livre-ier-des-personnes/titre-xi-de-la-majorite-et-des-majeurs-proteges-par-la-loi/chapitre-ii-des-mesures-de-protection-juridique-des-majeurs/section-4-de-la-curatelle-et-de-la-tutelle/sous-section-1-de-la-duree-de-la-mesure/442.md'>articles 442 </a>et <a href='/code-civil/livre-ier-des-personnes/titre-xi-de-la-majorite-et-des-majeurs-proteges-par-la-loi/chapitre-ii-des-mesures-de-protection-juridique-des-majeurs/section-5-du-mandat-de-protection-future/sous-section-1-des-dispositions-communes/485.md'>485</a> du code civil, le juge est saisi par requête remise ou adressée au greffe du tribunal d'instance.</p>
