# Article 1262-6

Lorsque le juge statue en application du deuxième alinéa de <a href='/code-civil/livre-ier-des-personnes/titre-xi-de-la-majorite-et-des-majeurs-proteges-par-la-loi/chapitre-iii-de-la-mesure-daccompagnement-judiciaire/495-4.md'>l'article 495-4 </a>du code civil, les <a href='/code-de-procedure-civile/livre-iii-dispositions-particulieres-a-certaines-matieres/titre-ier-les-personnes/chapitre-xi-les-regimes-de-protection-des-majeurs/1262-3.md'>articles 1262-3 à 1262-5</a> du présent code sont applicables.
