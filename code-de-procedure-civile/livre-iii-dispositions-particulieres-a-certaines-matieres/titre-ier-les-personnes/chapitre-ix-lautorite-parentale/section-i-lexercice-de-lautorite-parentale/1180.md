# Article 1180

<p>Les demandes formées en application de <a href='/code-civil/livre-ier-des-personnes/titre-ix-de-lautorite-parentale/chapitre-ier-de-lautorite-parentale-relativement-a-la-personne-de-lenfant/371-4.md'>l'article 371-4 </a>et du premier alinéa de <a href='/code-civil/livre-ier-des-personnes/titre-ix-de-lautorite-parentale/chapitre-ier-de-lautorite-parentale-relativement-a-la-personne-de-lenfant/section-1-de-lexercice-de-lautorite-parentale/paragraphe-4-de-lintervention-des-tiers/373-3.md'>l'article 373-3 du</a> code civil obéissent aux règles de la procédure écrite ordinaire applicable devant le tribunal judiciaire ; elles sont jugées après avis du ministère public.</p>
