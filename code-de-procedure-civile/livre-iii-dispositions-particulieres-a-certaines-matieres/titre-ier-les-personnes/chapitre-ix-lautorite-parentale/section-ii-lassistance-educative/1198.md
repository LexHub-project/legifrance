# Article 1198

Le juge peut visiter ou faire visiter tout mineur faisant l'objet d'une mesure de placement prise en application des <a href='/code-civil/livre-ier-des-personnes/titre-ix-de-lautorite-parentale/chapitre-ier-de-lautorite-parentale-relativement-a-la-personne-de-lenfant/section-2-de-lassistance-educative/375-3.md' title='Code civil - art. 375-3 (V)'>articles 375-3 et 375-5</a> du code civil.
