# Article L512-6

<p>Les personnes morales déclarées responsables pénalement, dans les conditions prévues par l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>article 121-2 du code pénal</a>, des infractions définies aux <a href='/code-minier-nouveau/partie-legislative/livre-v-infractions-et-sanctions-penales/titre-unique/chapitre-ii-sanctions-penales/section-1-dispositions-communes/l512-1.md' title='Code minier (nouveau) - art. L512-1 (VT)'>articles L. 512-1, L. 512-2 </a>et <a href='/code-minier-nouveau/partie-legislative/livre-v-infractions-et-sanctions-penales/titre-unique/chapitre-ii-sanctions-penales/section-1-dispositions-communes/l512-5.md' title='Code minier (nouveau) - art. L512-5 (VT)'>L. 512-5 </a>du présent code encourent, outre l'amende suivant les modalités prévues par l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>article 131-38 du code pénal</a>, les peines prévues par les 2° à 6°, 8° et 9° de l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md' title='Code pénal - art. 131-39 (V)'>131-39</a> de ce code. </p><p>L'interdiction mentionnée au 2° de l'article 131-39 du code pénal porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.</p>
