# Article L332-20

Les personnes morales reconnues pénalement responsables, dans les conditions prévues à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md' title='Code pénal - art. 121-2 (V)'>l'article 121-2 </a>du code pénal, des infractions définies par le présent chapitre encourent les peines suivantes : 1° L'amende dans les conditions prévues à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md' title='Code pénal - art. 131-38 (V)'>l'article 131-38 </a>du code pénal ; 2° Dans les cas prévus par les <a href='/code-du-sport/partie-legislative/livre-iii-pratique-sportive/titre-ier-lieux-de-pratiques-sportives/chapitre-ii-equipements-sportifs/section-4-dispositions-penales/l312-14.md' title='Code du sport. - art. L312-14 (V)'>articles L. 312-14, L. 312-15, L. 312-16, </a><a href='/code-du-sport/partie-legislative/livre-iii-pratique-sportive/titre-iii-manifestations-sportives/chapitre-ii-securite-des-manifestations-sportives/l332-8.md' title='Code du sport. - art. L332-8 (V)'>L. 332-8, L. 332-9, L. 332-10, L. 332-11 (deuxième alinéa) </a>et <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071318&idArticle=LEGIARTI000006547745&dateTexte=&categorieLien=cid' title='Code du sport. - art. L332-19 (V)'>L. 332-19 </a>du présent code, les peines mentionnées à l'article 131-39 du code pénal.L'interdiction mentionnée au 2° de <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md' title='Code pénal - art. 131-39 (V)'>l'article 131-39</a> du code pénal porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.
