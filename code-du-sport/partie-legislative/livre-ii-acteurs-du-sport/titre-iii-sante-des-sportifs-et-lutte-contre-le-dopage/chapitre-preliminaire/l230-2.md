# Article L230-2

<p>Pour l'application du présent titre, une manifestation sportive internationale est une manifestation sportive pour laquelle un organisme sportif international : </p><p> 1° Soit édicte les règles qui sont applicables à cette manifestation ; </p><p> 2° Soit nomme les personnes chargées de faire respecter les règles applicables à cette manifestation. </p><p> Constituent des organismes sportifs internationaux au sens du présent article : </p><p> 1° Le Comité international olympique ; </p><p> 2° Le Comité international paralympique ; </p><p> 3° Une fédération sportive internationale signataire du code mondial antidopage mentionné par la convention internationale contre le dopage dans le sport, adoptée à Paris le 19 octobre 2005 ; </p><p> 4° Une organisation responsable d'une grande manifestation sportive internationale signataire du code mondial antidopage mentionné par la convention internationale contre le dopage dans le sport, adoptée à Paris le 19 octobre 2005. </p>
