# Article R1412-5

<p>Lorsqu'un salarié est temporairement détaché sur le territoire national par une entreprise établie dans un autre Etat membre de l'Union européenne, les contestations relatives aux droits reconnus dans les matières énumérées à l'article <a href='/code-du-travail/partie-legislative/premiere-partie-les-relations-individuelles-de-travail/livre-ii-le-contrat-de-travail/titre-vi-salaries-detaches-temporairement-par-une-entreprise-non-etablie-en-france/chapitre-ii-conditions-de-detachement-et-reglementation-applicable/section-2-reglementation-applicable/l1262-4.md'>L. 1262-4</a> peuvent être portées devant le conseil de prud'hommes dans le ressort duquel la prestation est ou a été exécutée. <br/>Lorsque la prestation est ou a été exécutée dans le ressort de plusieurs conseils de prud'hommes, ces contestations sont portées devant l'une quelconque de ces juridictions.</p>
