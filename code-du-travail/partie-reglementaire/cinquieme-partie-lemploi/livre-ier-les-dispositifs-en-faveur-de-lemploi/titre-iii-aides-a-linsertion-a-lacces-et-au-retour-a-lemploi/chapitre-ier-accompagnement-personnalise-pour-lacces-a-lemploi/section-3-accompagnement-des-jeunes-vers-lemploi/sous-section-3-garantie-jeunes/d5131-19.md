# Article D5131-19

<p>I.-Le niveau de ressources ouvrant droit à la garantie jeunes, pour l'application de l'article <a href='/code-du-travail/partie-legislative/cinquieme-partie-lemploi/livre-ier-les-dispositifs-en-faveur-de-lemploi/titre-iii-aides-a-linsertion-a-lacces-et-au-retour-a-lemploi/chapitre-v-dispositions-penales/l5135-6.md'>L. 5131-6</a>, correspond au montant forfaitaire du revenu de solidarité active mentionné à l'article <a href='/code-de-laction-sociale-et-des-familles/partie-legislative/livre-ii-differentes-formes-daide-et-daction-sociales/titre-vi-lutte-contre-la-pauvrete-et-les-exclusions/chapitre-ii-revenu-minimum-dinsertion/section-2-conditions-douverture-du-droit-a-lallocation-et-prime-forfaitaire/sous-section-1-conditions-douverture-du-droit/l262-2.md'>L. 262-2 </a>du code de l'action sociale et des familles pour une personne seule, déduction faite de la fraction mentionnée au 1° de l'article <a href='/code-de-laction-sociale-et-des-familles/partie-reglementaire/livre-ii-differentes-formes-daide-et-daction-sociales/titre-vi-lutte-contre-la-pauvrete-et-les-exclusions/chapitre-ii-revenu-minimum-dinsertion/section-2-conditions-douverture-du-droit-a-lallocation-et-prime-forfaitaire/sous-section-1-determination-des-ressources/paragraphe-2-determination-des-ressources/r262-9.md'>R. 262-9 </a>du même code. </p><p>II.-Sont pris en compte pour la détermination du niveau de ressources ouvrant droit au bénéfice de la garantie jeunes : </p><p>1° Les revenus mentionnés aux articles <a href='/code-de-la-securite-sociale/partie-reglementaire-decrets-en-conseil-detat/livre-8-allocations-aux-personnes-agees-allocation-aux-adultes-handicapes-allocation-de-logement-sociale-aides-a-lemploi-pour-la-garde-des-jeunes-enfants-aide-aux-collectivites-et-organismes-logeant-a-titre-temporaire-des-personnes-defavorisees/titre-4-aides-a-lemploi-pour-la-garde-des-jeunes-enfants/chapitre-4-ressources-prises-en-compte-pour-la-prime-dactivite/r844-1.md'>R. 844-1 et R. 844-2 </a>du code de la sécurité sociale ; </p><p>2° Les bourses d'études ainsi que l'allocation pour la diversité dans la fonction publique ; </p><p>3° Les revenus tirés de stages réalisés en application de l'article <a href='/code-de-leducation/partie-legislative/premiere-partie-dispositions-generales-et-communes/livre-ier-principes-generaux-de-leducation/titre-ii-objectifs-et-missions-du-service-public-de-lenseignement/chapitre-iv-stages-et-periodes-de-formation-en-milieu-professionnel/l124-1.md'>L. 124-1 </a>du code de l'éducation ; </p><p>4° L'allocation aux adultes handicapés mentionnée aux articles <a href='/code-de-la-securite-sociale/partie-legislative/livre-8-allocations-aux-personnes-agees-allocation-aux-adultes-handicapes-allocation-de-logement-sociale-aides-a-lemploi-pour-la-garde-des-jeunes-enfants-aides-aux-collectivites-et-organismes-logeant-a-titre-temporaire-des-personnes-defavorisee/titre-ii-allocation-aux-adultes-handicapes/l821-1.md'>L. 821-1 et L. 821-2 </a>du code de la sécurité sociale ; </p><p>5° L'allocation temporaire d'attente mentionnée à l'article <a href='/code-du-travail/partie-legislative/cinquieme-partie-lemploi/livre-iv-le-demandeur-demploi/titre-ii-indemnisation-des-travailleurs-involontairement-prives-demploi/chapitre-iii-regime-de-solidarite/section-1-allocations/sous-section-3-allocation-temporaire-dattente/l5423-8.md'>L. 5423-8 </a>du code du travail ; </p><p>6° Le revenu de solidarité active mentionné à l'article <a href='/code-de-laction-sociale-et-des-familles/partie-legislative/livre-ii-differentes-formes-daide-et-daction-sociales/titre-vi-lutte-contre-la-pauvrete-et-les-exclusions/chapitre-ii-revenu-minimum-dinsertion/section-1-dispositions-generales/l262-1.md'>L. 262-1</a> du code de l'action sociale et des familles ; </p><p>7° La prime d'activité mentionnée à l'article <a href='/code-de-la-securite-sociale/partie-legislative/livre-8-allocations-aux-personnes-agees-allocation-aux-adultes-handicapes-allocation-de-logement-sociale-aides-a-lemploi-pour-la-garde-des-jeunes-enfants-aides-aux-collectivites-et-organismes-logeant-a-titre-temporaire-des-personnes-defavorisee/titre-4-aides-a-lemploi-pour-la-garde-des-jeunes-enfants/chapitre-1-aide-a-la-famille-pour-lemploi-dune-assistante-maternelle-agreee/l841-1.md'>L. 841-1 </a>du code de la sécurité sociale.<br/></p>
