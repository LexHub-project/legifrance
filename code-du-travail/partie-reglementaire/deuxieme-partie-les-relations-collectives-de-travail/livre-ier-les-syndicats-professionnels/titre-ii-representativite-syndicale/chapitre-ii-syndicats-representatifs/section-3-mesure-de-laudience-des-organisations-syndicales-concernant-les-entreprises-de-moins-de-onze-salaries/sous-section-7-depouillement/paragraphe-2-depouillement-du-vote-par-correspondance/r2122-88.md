# Article R2122-88

<p>N'entrent pas en compte dans le résultat des votes par correspondance : </p><p>1° Les enveloppes sans bulletin ; </p><p>2° Les bulletins blancs ; </p><p>3° Les bulletins multiples trouvés dans la même enveloppe et en faveur de candidatures différentes ; </p><p>4° Les bulletins désignant une candidature qui n'a pas été régulièrement publiée ou dont l'irrecevabilité a été constatée par le juge ; </p><p>5° Les bulletins d'un modèle différent de ceux qui ont été adressés aux électeurs ou qui comportent une mention manuscrite rendant incertaine l'expression du vote ; </p><p>6° Les bulletins portant des signes intérieurs ou extérieurs de reconnaissance ; </p><p>7° Les bulletins portant des mentions injurieuses ; </p><p>8° Les documents de propagande utilisés comme bulletin. </p>
