# Article L8211-1

<p>Sont constitutives de travail illégal, dans les conditions prévues par le présent livre, les infractions suivantes : </p><p>1° Travail dissimulé ; </p><p>2° Marchandage ; </p><p>3° Prêt illicite de main-d'oeuvre ; </p><p>4° Emploi d'étranger non autorisé à travailler ; </p><p>5° Cumuls irréguliers d'emplois ; </p><p>6° Fraude ou fausse déclaration prévue aux <a href='/code-du-travail/partie-legislative/cinquieme-partie-lemploi/livre-ier-les-dispositifs-en-faveur-de-lemploi/titre-ii-aides-au-maintien-et-a-la-sauvegarde-de-lemploi/chapitre-iv-dispositions-penales/l5124-1.md'>articles L. 5124-1 </a>et <a href='/code-du-travail/partie-legislative/cinquieme-partie-lemploi/livre-iv-le-demandeur-demploi/titre-ii-indemnisation-des-travailleurs-involontairement-prives-demploi/chapitre-ix-dispositions-penales/l5429-1.md'>L. 5429-1</a>.</p><br/><br/><i>NOTA:<p><br clear='none'/></p></i>
