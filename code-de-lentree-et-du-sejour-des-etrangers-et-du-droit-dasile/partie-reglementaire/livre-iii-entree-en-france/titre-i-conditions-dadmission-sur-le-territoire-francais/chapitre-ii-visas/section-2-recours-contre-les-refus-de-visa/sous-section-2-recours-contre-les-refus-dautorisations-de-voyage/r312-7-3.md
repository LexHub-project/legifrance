# Article R312-7-3

<p>La personne dont le recours a été rejeté par le sous-directeur des visas peut demander au tribunal administratif l'annulation de cette décision dans les délais prévus par les articles <a href='/code-de-justice-administrative/partie-reglementaire-decrets-en-conseil-detat/livre-iv-lintroduction-de-linstance-de-premier-ressort/titre-ii-les-delais/r421-1.md'>R. 421-1 </a>et <a href='/code-de-justice-administrative/partie-reglementaire-decrets-en-conseil-detat/livre-iv-lintroduction-de-linstance-de-premier-ressort/titre-ii-les-delais/r421-2.md'>R. 421-2 </a>du code de justice administrative. Les <a href='/code-de-justice-administrative/partie-reglementaire-decrets-en-conseil-detat/livre-iv-lintroduction-de-linstance-de-premier-ressort/titre-ii-les-delais/r421-7.md'>dispositions de l'article R. 421-7 du code de justice administrative </a>ne sont pas applicables.</p>
