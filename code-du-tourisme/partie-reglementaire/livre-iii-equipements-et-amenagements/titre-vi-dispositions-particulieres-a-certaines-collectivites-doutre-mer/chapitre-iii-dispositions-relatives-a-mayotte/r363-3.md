# Article R363-3

<p>Pour l'application de l'article <a href='/code-du-tourisme/partie-reglementaire/livre-iii-equipements-et-amenagements/titre-iii-terrains-de-camping-ou-de-caravanage-et-autres-terrains-amenages/chapitre-ii-classement/section-2-procedure-de-classement/d332-6.md'>D. 332-6</a>, les références aux <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071514&idArticle=LEGIARTI000006611020&dateTexte=&categorieLien=cid'>articles L. 411-1 et R. 412-16 du code forestier </a>sont respectivement remplacées par celles aux articles <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071556&idArticle=LEGIARTI000006614696&dateTexte=&categorieLien=cid'>L. 411-1 </a>et <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071556&idArticle=LEGIARTI000006614894&dateTexte=&categorieLien=cid'>R. * 412-14 </a>du code forestier de Mayotte. </p><p>Pour l'application de l'article <a href='/code-du-tourisme/partie-reglementaire/livre-iii-equipements-et-amenagements/titre-iii-terrains-de-camping-ou-de-caravanage-et-autres-terrains-amenages/chapitre-ii-classement/section-2-procedure-de-classement/d332-9.md'>D. 332-9</a>, la référence au <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071514&idArticle=LEGIARTI000006613103&dateTexte=&categorieLien=cid'>2° de l'article R. 412-17 du code forestier </a>est remplacée par celle au dernier alinéa de l'article <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071556&idArticle=LEGIARTI000006614897&dateTexte=&categorieLien=cid'>R. * 412-15</a> du code forestier de Mayotte.</p>
