# Article L163-10

Les <a href='/code-du-tourisme/partie-legislative/livre-ier-organisation-generale-du-tourisme/titre-iii-les-collectivites-territoriales-et-leurs-groupements/chapitre-3-la-commune/section-1-organismes-communaux-de-tourisme/sous-section-1-dispositions-communes-applicables-aux-offices-de-tourisme/l133-1.md'>articles L. 133-1 à L. 141-1 </a>sont applicables à Mayotte dans les conditions suivantes : 1° Pour l'application de <a href='/code-du-tourisme/partie-legislative/livre-ier-organisation-generale-du-tourisme/titre-iii-les-collectivites-territoriales-et-leurs-groupements/chapitre-4-groupements-intercommunaux/section-1-dispositions-generales/l134-1.md'>l'article L. 134-1</a>, la référence à l'article L. 5215-20-1 du code général des collectivités territoriales n'est pas applicable à Mayotte et <a href='/code-general-des-collectivites-territoriales/partie-legislative/cinquieme-partie-la-cooperation-locale/livre-ii-la-cooperation-intercommunale/titre-ier-etablissements-publics-de-cooperation-intercommunale/chapitre-vi-communaute-dagglomeration/section-4-competences/l5216-5.md'>l'article L. 5216-5 </a>du même code est applicable à Mayotte dans les conditions prévues à l'article L. 5832-21 du même code ; 2° Pour l'application de <a href='/code-du-tourisme/partie-legislative/livre-ier-organisation-generale-du-tourisme/titre-iii-les-collectivites-territoriales-et-leurs-groupements/chapitre-4-groupements-intercommunaux/section-1-dispositions-generales/l134-2.md'>l'article L. 134-2, </a>le 2° du I de <a href='/code-general-des-collectivites-territoriales/partie-legislative/cinquieme-partie-la-cooperation-locale/livre-ii-la-cooperation-intercommunale/titre-ier-etablissements-publics-de-cooperation-intercommunale/chapitre-iv-communaute-de-communes/section-4-competences/l5214-16.md'>l'article L. 5214-16 </a>du code général des collectivités territoriales est applicable à compter de la date d'entrée en vigueur à Mayotte du code général des impôts prévue par <a href='/affichTexteArticle.do?cidTexte=JORFTEXT000000407771&idArticle=LEGIARTI000006386580&dateTexte=&categorieLien=cid'>l'article 68</a> de la loi n° 2001-616 du 11 juillet 2001 relative à Mayotte.
