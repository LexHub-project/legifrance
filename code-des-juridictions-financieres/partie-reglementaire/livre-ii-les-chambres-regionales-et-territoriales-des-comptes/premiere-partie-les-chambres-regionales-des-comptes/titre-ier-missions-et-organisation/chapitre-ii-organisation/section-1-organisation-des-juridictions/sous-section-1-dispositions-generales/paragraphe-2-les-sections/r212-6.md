# Article R212-6

<p>Le nombre des sections des chambres régionales des comptes est fixé comme suit : </p><p> Alsace : une section ; </p><p> Aquitaine, Poitou-Charentes : quatre sections ; </p><p> Auvergne, Rhône-Alpes : cinq sections ; </p><p> Basse-Normandie, Haute-Normandie : deux sections ; </p><p> Bourgogne, Franche-Comté : deux sections ; </p><p> Bretagne : trois sections ; </p><p> Centre, Limousin : trois sections ; </p><p> Champagne-Ardenne, Lorraine : trois sections ; </p><p> Guadeloupe : une section ; </p><p> Guyane : une section ; </p><p> Ile-de-France : huit sections ; </p><p> Languedoc-Roussillon : deux sections ; </p><p> Martinique : une section ; </p><p> Midi-Pyrénées : deux sections ; </p><p> Nord-Pas-de-Calais, Picardie : quatre sections ; </p><p> Pays de la Loire : trois sections ; </p><p> Provence-Alpes-Côte d'Azur : quatre sections. </p>
