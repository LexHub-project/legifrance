# Article R212-1

<p>Les sièges et les ressorts des chambres régionales des comptes sont fixés comme suit : </p><p> Arras : Nord-Pas-de-Calais, Picardie ; </p><p> Bastia : Corse ; </p><p>  Bordeaux : Aquitaine, Limousin, Poitou-Charentes ;</p><p> Cayenne : Guyane ; </p><p> Dijon : Bourgogne, Franche-Comté ; </p><p> Fort-de-France : Martinique ; </p><p> Lyon : Auvergne, Rhône-Alpes ; </p><p> Mamoudzou : Mayotte ; </p><p> Marseille : Provence-Alpes-Côte d'Azur ; </p><p>Metz : Alsace, Champagne-Ardenne, Lorraine ; </p><p> Montpellier : Languedoc-Roussillon, Midi-Pyrénées ;</p><p> Nantes : Pays de la Loire ; </p><p> Noisiel : Ile-de-France ; </p><p> Orléans : Centre Val de Loire ; </p><p> Pointe-à-Pitre : Guadeloupe ; </p><p> Rennes : Bretagne ; </p><p> Rouen : Normandie ;</p><p> Saint-Denis : La Réunion.</p><br/>
