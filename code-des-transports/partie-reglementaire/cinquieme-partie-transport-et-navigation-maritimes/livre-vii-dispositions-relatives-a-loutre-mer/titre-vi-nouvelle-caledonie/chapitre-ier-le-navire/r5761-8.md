R5141-14 (V)'>R. 5141-14 </a>et <a href='/code-des-transports/partie-reglementaire/cinquieme-partie-transport-et-navigation-maritimes/livre-ier-le-navire/titre-iv-navires-abandonnes-et-epaves/chapitre-ii-epaves/section-1-dispositions-generales/sous-section-2-vente-ou-concession-des-epaves/r5142-13.md' title='Code des transports - art. R5142-13 (V)'>R. 5142-13</a>, les mots : " code général de la propriété des personnes publiques " sont remplacés par les mots : " code du domaine de l'Etat ".
