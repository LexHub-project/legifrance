# Article L5111-1

<p>Les éléments d'identification des navires sont : <br/>1° Le nom, indiqué par le certificat d'immatriculation ; <br/>2° Le port d'attache ; <br/>3° La nationalité ; <br/>4° Le tonnage défini en unités de jauge en application de l'article <a href='/code-des-transports/partie-legislative/cinquieme-partie-transport-et-navigation-maritimes/l5000-5.md' title='Code des transports - art. L5000-5 (V)'>L. 5000-5</a> du présent code. <br/>Des marques extérieures d'identification doivent être portées sur les navires dans les conditions définies par voie réglementaire.</p>
