# Article L2135-11

<p>Les personnes morales déclarées responsables pénalement, dans les conditions prévues par <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-1-des-peines-applicables-aux-personnes-physiques/sous-section-1-des-peines-criminelles/131-2.md'>l'article 121-2 </a>du code pénal, de l'infraction définie à <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000023086525&idArticle=LEGIARTI000023084419&dateTexte=&categorieLien=id' title='Code des transports - art. L2135-9 (Ab)'>l'article L. 2135-9 </a>encourent, outre l'amende, suivant les modalités prévues par <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>l'article 131-38 </a>du code pénal, les peines prévues par les 2°, 4° et 9° de <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>l'article 131-39</a> de ce code. L'interdiction mentionnée au 2° de l'article 131-39 du code pénal porte sur l'activité professionnelle ou sociale dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.</p>
