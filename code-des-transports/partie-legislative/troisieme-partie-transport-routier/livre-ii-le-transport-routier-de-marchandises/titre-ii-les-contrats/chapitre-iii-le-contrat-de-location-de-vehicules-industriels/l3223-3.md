# Article L3223-3

<p>Les articles <a href='/code-des-transports/partie-legislative/troisieme-partie-transport-routier/livre-ii-le-transport-routier-de-marchandises/titre-ii-les-contrats/chapitre-ii-le-contrat-de-transport/l3222-1.md'>L. 3222-1 </a>et <a href='/code-des-transports/partie-legislative/troisieme-partie-transport-routier/livre-ii-le-transport-routier-de-marchandises/titre-ii-les-contrats/chapitre-ii-le-contrat-de-transport/l3222-2.md'>L. 3222-2 </a>sont applicables aux contrats de location de véhicules avec conducteur destinés au transport routier de marchandises. </p><p>L'article <a href='/code-des-transports/partie-legislative/troisieme-partie-transport-routier/livre-ii-le-transport-routier-de-marchandises/titre-ii-les-contrats/chapitre-ii-le-contrat-de-transport/l3222-3.md'>L. 3222-3 </a>est applicable à ces contrats de location lorsque le loueur est le redevable destinataire des avis de paiement des taxes prévues aux articles <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071570&idArticle=LEGIARTI000006615268&dateTexte=&categorieLien=cid'>269 </a>à <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006071570&idArticle=LEGIARTI000020033028&dateTexte=&categorieLien=cid'>283 quater</a> du code des douanes.</p>
