# Article L641-3

<p>Pour l'application du livre III de la partie législative du code à Saint-Pierre-et-Miquelon :</p><p>1° Au 1° de l'article <a href='/code-de-lexpropriation-pour-cause-dutilite-publique/partie-legislative-nouvelle/livre-iii-indemnisation/titre-ii-fixation-et-paiement-des-indemnites/chapitre-ii-modalites-devaluation-de-lindemnite-dexpropriation/l322-3.md' title='Code de l'expropriation pour cause d'utilité pu... - art. L322-3 (VD)'>L. 322-3</a>, les mots : " situés dans une partie actuellement urbanisée d'une commune " sont remplacés par les mots : " situés soit dans une partie actuellement urbanisée d'une commune, soit dans une partie de commune désignée comme constructible par le conseil territorial de Saint-Pierre-et-Miquelon " ;</p><p>2° L'article <a href='/code-de-lexpropriation-pour-cause-dutilite-publique/partie-legislative-nouvelle/livre-iii-indemnisation/titre-ii-fixation-et-paiement-des-indemnites/chapitre-ii-modalites-devaluation-de-lindemnite-dexpropriation/l322-8.md' title='Code de l'expropriation pour cause d'utilité pu... - art. L322-8 (VD)'>L. 322-8</a> est ainsi modifié :</p><p>a) Le deuxième alinéa est supprimé ;</p><p>b) Au troisième alinéa, le mot : " il " est remplacé par les mots : " le juge ".</p>
