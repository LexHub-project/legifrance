# Article L31

Hors les cas prévus par les conventions internationales, est interdite, sous les peines prévues par <a href='/code-penal/partie-legislative/livre-ii-des-crimes-et-delits-contre-les-personnes/titre-ii-des-atteintes-a-la-personne-humaine/chapitre-ii-des-atteintes-a-lintegrite-physique-ou-psychique-de-la-personne/section-4-du-trafic-de-stupefiants/222-36.md' title='Code pénal - art. 222-36 (V)'>l'article 222-36</a> du code pénal, l'insertion dans les envois postaux d'opium, de morphine, de cocaïne et autres stupéfiants.
