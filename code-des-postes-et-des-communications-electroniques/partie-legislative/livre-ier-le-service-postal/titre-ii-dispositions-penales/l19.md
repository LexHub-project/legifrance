# Article L19

<p>Les personnes morales déclarées responsables pénalement, dans les conditions prévues par l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>article 121-2 du code pénal</a>, de l'infraction définie à <a href='/code-des-postes-et-des-communications-electroniques/partie-legislative/livre-ier-le-service-postal/titre-ii-dispositions-penales/l17.md'>l'article L. 17</a> encourent, outre l'amende suivant les modalités prévues par l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>article 131-38 du code pénal</a>, les peines prévues par les 2° à 5°, 8° et 9° de l'article 131-39 du même code.L'interdiction mentionnée au 2° de l'<a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>article 131-39 </a>du même code porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.</p>
