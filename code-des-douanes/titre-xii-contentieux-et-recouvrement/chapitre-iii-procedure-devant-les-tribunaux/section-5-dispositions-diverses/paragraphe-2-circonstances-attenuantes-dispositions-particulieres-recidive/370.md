# Article 370

1. Si le contrevenant aux dispositions des articles <a href='/code-des-douanes/titre-xii-contentieux-et-recouvrement/chapitre-vi-dispositions-repressives/section-1-classification-des-infractions-douanieres-et-peines-principales/paragraphe-2-contraventions-douanieres/a-premiere-classe/410.md' title='Code des douanes - art. 410 (V)'>410,411,412,414</a> du présent code commet dans les cinq ans qui suivent une transaction ou une condamnation devenue définitive, une nouvelle infraction tombant sous le coup des sanctions prévues par les articles précités, le taux maximal des pénalités encourues est doublé. 2. Cette disposition n'est pas applicable, sauf le cas de faute personnelle, à ceux qui font profession d'accomplir pour autrui les formalités de douane.
