# Article R533-2

<p>La publicité définitive est opérée, pour l'hypothèque, conformément à l'<a href='/code-civil/livre-iv-des-suretes/titre-ii-des-suretes-reelles/sous-titre-iii-des-suretes-sur-les-immeubles/chapitre-iii-des-hypotheques/section-6-de-linscription-des-hypotheques/sous-section-1-du-mode-dinscription-des-hypotheques/2428.md'>article 2428 du code civil </a>et, pour le nantissement du fonds de commerce, conformément aux articles <a href='/code-de-commerce/partie-legislative/livre-ier-du-commerce-en-general/titre-iv-du-fonds-de-commerce/chapitre-iii-dispositions-communes-a-la-vente-et-au-nantissement-du-fonds-de-commerce/section-2-des-formalites-dinscription-et-de-radiation/l143-16.md'>L. 143-16</a> et <a href='/code-de-commerce/partie-reglementaire/livre-v-des-effets-de-commerce-et-des-garanties/titre-ii-des-garanties/chapitre-ier-dispositions-generales-sur-le-gage-commercial/section-1-contenu-et-forme-electronique-du-registre/r521-1.md'>R. 521-1 </a>et suivants du code de commerce. </p><p>Il n'est dû qu'un seul émolument ou qu'une seule contribution de sécurité immobilière prévue à <a href='/code-general-des-impots-cgi/livre-premier-assiette-et-liquidation-de-limpot/premiere-partie-impots-detat/titre-iv-enregistrement-publicite-fonciere-impot-de-solidarite-sur-la-fortune-timbre/chapitre-premier-droits-denregistrement-et-taxe-de-publicite-fonciere/section-iii-obligations-diverses/ii-obligations-des-agents-de-ladministration/879.md'>l'article 879 </a>du code général des impôts pour les inscriptions provisoire et définitive.</p>
