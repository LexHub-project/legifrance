# Article R241-1

<p>Les dispositions particulières relatives aux autres procédures d'exécution mobilière sont énoncées : </p><p><br/>1° Par le <a href='/affichTexte.do?cidTexte=JORFTEXT000000702914&categorieLien=cid' title='Décret n°67-967 du 27 octobre 1967 (V)'>décret n° 67-967 du 27 octobre 1967 </a>relatif au statut des navires et autres bâtiments de mer pour la saisie des navires ; </p><p><br/>2° Par le code de l'aviation civile pour la saisie des aéronefs ; </p><p><br/>3° Par le code du domaine public fluvial et de la navigation intérieure pour la saisie des bateaux ; </p><p><br/>4° Par le code de la propriété intellectuelle pour la saisie en matière de droit de propriété littéraire, artistique et industrielle ; </p><p><br/>5° Par le code rural et de la pêche maritime pour les oppositions à tiers détenteur des mutualités sociales agricoles ; </p><p><br/>6° Par le code de la sécurité sociale pour les oppositions à tiers détenteur des caisses de sécurité sociale.<br/></p>
