# Article R314-67-2

<p>La demande d'émission de garanties d'origine portant sur l'électricité autoconsommée comporte les mêmes éléments que ceux figurant aux articles <a href='/code-de-lenergie/partie-reglementaire/livre-iii-les-dispositions-relatives-a-lelectricite/titre-ier-la-production/chapitre-iv-les-dispositions-particulieres-a-lelectricite-produite-a-partir-denergies-renouvelables/section-2-les-garanties-dorigine/sous-section-3-emission-transfert-et-annulation-des-garanties-dorigine/r314-60.md'>R. 314-60 </a>et <a href='/code-de-lenergie/partie-reglementaire/livre-iii-les-dispositions-relatives-a-lelectricite/titre-ier-la-production/chapitre-iv-les-dispositions-particulieres-a-lelectricite-produite-a-partir-denergies-renouvelables/section-2-les-garanties-dorigine/sous-section-3-emission-transfert-et-annulation-des-garanties-dorigine/r314-61.md'>R. 314-61</a>. Dans le cas où l'installation de production n'est pas directement raccordée au réseau public d'électricité, le producteur indique, le cas échéant, le nom du gestionnaire de réseau de son site de consommation.</p>
