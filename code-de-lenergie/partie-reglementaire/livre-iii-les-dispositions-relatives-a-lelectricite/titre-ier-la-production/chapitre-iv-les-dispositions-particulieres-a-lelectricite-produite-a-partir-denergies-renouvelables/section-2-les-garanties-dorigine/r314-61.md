# Article R314-61

<p>La demande indique également : </p><p>1° Lorsque l'électricité a été produite à partir de sources d'énergie renouvelables : </p><p>a) La nature de la source d'énergie à partir de laquelle l'électricité a été produite ; </p><p>b) La part d'électricité produite à partir d'énergies renouvelables, si l'installation fonctionne avec d'autres sources d'énergie ; </p><p>2° Lorsque l'électricité est produite par cogénération : </p><p>a) La puissance thermique de l'installation ; </p><p>b) Les combustibles à partir desquels l'électricité a été produite ; </p><p>c) Le pouvoir calorifique inférieur des combustibles utilisés ; </p><p>d) Le rendement global de l'installation ; </p><p>e) La quantité de chaleur produite au cours de la période pour laquelle la garantie d'origine est demandée ; </p><p>f) L'utilisation de la chaleur produite en même temps que l'électricité ; </p><p>g) Les économies d'énergie primaire réalisées, calculées conformément aux dispositions des arrêtés prévus à l'article <a href='/code-de-lenergie/partie-reglementaire/livre-iii-les-dispositions-relatives-a-lelectricite/titre-ier-la-production/chapitre-iv-les-dispositions-particulieres-a-lelectricite-produite-a-partir-denergies-renouvelables/section-2-les-garanties-dorigine/r314-56.md' title='Code de l'énergie - art. R314-56 (V)'>R. 314-56</a>.</p>
