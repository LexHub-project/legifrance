# Article D723-255

<p>Les catégories d'informations qui peuvent être demandées dans les conditions prévues par <a href='/code-rural-et-de-la-peche-maritime/partie-reglementaire/livre-vii-dispositions-sociales/titre-ii-organisation-generale-des-regimes-de-protection-sociale-des-professions-agricoles/chapitre-iii-organismes-de-protection-sociale-des-professions-agricoles/section-5-transmission-de-donnees/d723-254.md'>l'article D. 723-254</a> sont les suivantes : </p><p>1° Pour les informations portant sur les cotisants salariés et non salariés agricoles, ainsi que sur les membres des familles des non-salariés : </p><p>a) Un numéro d'identifiant anonyme ; </p><p>b) L'année de naissance ; </p><p>c) Le sexe ; </p><p>d) La situation familiale ; </p><p>e) La nationalité (française, Union européenne ou hors Union européenne) ; </p><p>2° Pour les informations concernant les entreprises et les établissements relevant des régimes de protection sociale agricole : </p><p>a) La forme juridique ; </p><p>b) La date de création ; </p><p>c) Le statut des associés pour les non-salariés ; </p><p>d) La description de l'activité ; </p><p>e) Le département du siège social ; </p><p>f) Les caractéristiques de gestion (périodicité de versement des cotisations, classe de risque en législation accidents du travail...) ; </p><p>g) Les effectifs ; </p><p>h) L'identifiant de convention collective. </p><p>3° Pour les informations sur l'emploi et l'activité des salariés : </p><p>a) La date de début et de fin du contrat de travail ; </p><p>b) La durée de travail ; </p><p>c) La nature et la qualité du contrat ; </p><p>d) Le statut de l'emploi ; </p><p>e) La description des activités ; </p><p>f) Le montant de la rémunération ; </p><p>g) Les caractéristiques de gestion (type de déclaration d'embauche, classe de risque en législation accidents du travail...). </p><p>4° Pour les informations sur les cotisations et les éléments constitutifs et de calcul des assiettes de cotisations : </p><p>a) Qualité du cotisant non salarié agricole et branche d'affiliation ; </p><p>b) Les montants, taux et assiettes de cotisations par branche ; </p><p>c) La nature et les montants des réductions de cotisations ou d'abattements d'assiette ; </p><p>d) Les revenus professionnels et cadastraux des non-salariés agricoles ; </p><p>e) Le régime d'imposition des revenus professionnels des non salariés agricoles ; </p><p>f) Le type d'assiette des non-salariés agricoles ; </p><p>g) L'assiette brute et les éléments de calcul de l'assiette nette des non salariés agricoles. </p><p>5° Pour les informations visées aux 1° à 4°, le code de la nomenclature des activités françaises (NAF) pour les entreprises concernées.</p>
