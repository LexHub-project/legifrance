# Article R444-71

<p>Les dispositions de la présente sous-section sont applicables aux tarifs relatifs aux prestations de postulation des avocats dans les matières suivantes : </p><p>1° La saisie immobilière régie par les articles <a href='/code-des-procedures-civiles-dexecution/partie-legislative/livre-iii-la-saisie-immobiliere/titre-ier-dispositions-generales/chapitre-unique/l311-1.md'>L. 311-1 à L. 341-1 </a>et <a href='/code-des-procedures-civiles-dexecution/partie-reglementaire/livre-iii-la-saisie-immobiliere/titre-ier-dispositions-generales/chapitre-unique/r311-1.md'>R. 311-1 </a>à <a href='/code-des-procedures-civiles-dexecution/partie-reglementaire/livre-iii-la-saisie-immobiliere/titre-iii-la-distribution-du-prix/chapitre-iv-dispositions-communes/r334-3.md'>R. 334-3 </a>du code des procédures civiles d'exécution ; </p><p>2° Le partage régi par les articles <a href='/code-civil/livre-iii-des-differentes-manieres-dont-on-acquiert-la-propriete/titre-ier-des-successions/chapitre-vii-du-regime-legal-de-lindivision/815.md'>815 à 892 </a>du code civil et les articles <a href='/code-de-procedure-civile/livre-iii-dispositions-particulieres-a-certaines-matieres/titre-iii-les-regimes-matrimoniaux-les-successions-et-les-liberalites/chapitre-ii-les-successions-et-les-liberalites/section-vi-le-partage/sous-section-i-le-partage-amiable/1358.md'>1358 à 1376 </a>du code de procédure civile ; </p><p>3° La licitation régie par les articles 1686 à 1688 du code civil et les articles <a href='/code-de-procedure-civile/livre-iii-dispositions-particulieres-a-certaines-matieres/titre-iii-les-regimes-matrimoniaux-les-successions-et-les-liberalites/chapitre-ii-les-successions-et-les-liberalites/section-vi-le-partage/sous-section-ii-le-partage-judiciaire/paragraphe-3-la-licitation/1377.md'>1377 et 1378 </a>du code de procédure civile ; </p><p>4° Les sûretés judiciaires régies par les articles <a href='/code-des-procedures-civiles-dexecution/partie-legislative/livre-v-les-mesures-conservatoires/titre-iii-les-suretes-judiciaires/chapitre-ier-dispositions-generales/l531-1.md'>L. 531-1 à L. 533-1 </a>et <a href='/code-des-procedures-civiles-dexecution/partie-reglementaire/livre-v-les-mesures-conservatoires/titre-iii-les-suretes-judiciaires/chapitre-ier-dispositions-generales/r531-1.md'>R. 531-1 </a>à <a href='/code-des-procedures-civiles-dexecution/partie-reglementaire/livre-v-les-mesures-conservatoires/titre-iii-les-suretes-judiciaires/chapitre-iv-dispositions-applicables-aux-departements-du-bas-rhin-du-haut-rhin-et-de-la-moselle/r534-1.md'>R. 534-1 </a>du code des procédures civiles d'exécution et l'hypothèque légale régie par l'<a href='/code-civil/livre-iv-des-suretes/titre-ii-des-suretes-reelles/sous-titre-iii-des-suretes-sur-les-immeubles/chapitre-iii-des-hypotheques/section-2-des-hypotheques-legales/sous-section-1-dispositions-generales/paragraphe-3-des-regles-particulieres-a-lhypotheque-legale-attachee-aux-jugements-de-condamnation/2401.md'>article 2401 du code civil</a>.</p><br/><br/><i>NOTA:<p>Conformément au premier alinéa du I de l'article 7 du décret n° 2021-1888 du 29 décembre 2021, ces dispositions entrent en vigueur le 1er janvier 2022.</p></i>
