# Article L954-2-1

<p>Au dernier alinéa du III de <a href='/code-de-commerce/partie-legislative/livre-iv-de-la-liberte-des-prix-et-de-la-concurrence/titre-iv-de-la-transparence-des-pratiques-restrictives-de-concurrence-et-dautres-pratiques-prohibees/chapitre-preliminaire-dispositions-generales/l440-1.md'>l'article L. 440-1, </a>les mots : " du présent code ou à <a href='/code-de-la-consommation/partie-legislative/livre-ii-conformite-et-securite-des-produits-et-des-services/titre-ier-conformite/chapitre-v-pouvoirs-denquete/section-1-autorites-qualifiees/l215-1.md'>l'article L. 215-1</a> du code de la consommation " sont supprimés.</p>
