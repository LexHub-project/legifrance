# Article 1804 C

La méconnaissance de l'obligation prévue à <a href='/code-general-des-impots-cgi/livre-ii-recouvrement-de-limpot/chapitre-premier-paiement-de-limpot/section-iii-contributions-indirectes-1/1698-d.md' title='Code général des impôts, CGI. - art. 1698 D (V)'>l'article 1698 D</a> entraîne l'application d'une majoration de 0,2 % du montant des sommes dont le versement a été effectué selon un autre moyen de paiement. Cette majoration est recouvrée selon les procédures et sous le bénéfice des sûretés prévues par le présent code en matière de contributions indirectes.
