# Article R720-15

<p>Pour l'application de la partie réglementaire du code dans la collectivité territoriale de Saint-Pierre-et-Miquelon, les termes énumérés ci-après sont remplacés comme suit :</p><p>a) Les mots : " tribunal de grande instance " par les mots : " tribunal de première instance " ;</p><p>b) Les mots : " cour d'appel " par les mots : " tribunal supérieur d'appel " ;</p><p>c) Les mots : " département " ou " région " par le mot : " collectivité " ;</p><p>d) Les mots : " préfet " ou " préfet de région " par les mots : " représentant de l'Etat dans la collectivité " ;</p><p>e) Les mots : " arrêté préfectoral " par les mots : " arrêté du représentant de l'Etat dans la collectivité ".</p>
