# Article L760-5

<p>Pour l'application des <a href='/code-du-patrimoine/partie-legislative/livre-vii-dispositions-relatives-a-loutre-mer/titre-vi-dispositions-applicables-dans-les-iles-wallis-et-futuna/l760-1.md' title='Code du patrimoine - art. L760-1 (V)'>articles L. 760-1 à L. 760-4</a> dans les îles Wallis et Futuna, les termes énumérés ci-après sont remplacés ainsi qu'il suit : </p><p>a) Les mots : " département " ou " région " par le mot : " territoire " ; </p><p>b) Les mots : " cour d'appel " par les mots : " tribunal supérieur d'appel " ; </p><p>c) Le mot : " préfet " par les mots : " représentant de l'Etat dans le territoire ".</p>
