# Article L750-1

<p>I.-Pour l'application de l'article <a href='/code-du-patrimoine/partie-legislative/livre-ier-dispositions-communes-a-lensemble-du-patrimoine-culturel/titre-ier-protection-des-biens-culturels/chapitre-2-restitution-des-biens-culturels/section-3-action-en-revendication-et-action-en-nullite/l112-22.md'>L. 112-22</a>, les mots : “ les personnes mentionnées au troisième alinéa de l'article <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ier-regime-general-des-archives/chapitre-2-collecte-conservation-et-protection/section-1-archives-publiques/sous-section-1-dispositions-generales/l212-1.md'>L. 212-1 </a>” sont remplacés par les mots : “ l'Etat ou tout service public d'archives compétent ”.</p><p>Pour l'application de l'article <a href='/code-du-patrimoine/partie-legislative/livre-ier-dispositions-communes-a-lensemble-du-patrimoine-culturel/titre-ier-protection-des-biens-culturels/chapitre-2-restitution-des-biens-culturels/section-3-action-en-revendication-et-action-en-nullite/l112-23.md'>L. 112-23</a>, les mots : “ en application de l'article <a href='/code-civil/livre-iii-des-differentes-manieres-dont-on-acquiert-la-propriete/titre-vi-de-la-vente/chapitre-iv-des-obligations-du-vendeur/section-3-de-la-garantie/paragraphe-1-de-la-garantie-en-cas-deviction/1626.md'>1626</a> du code civil ” sont supprimés.</p><p>II.-Les articles <a href='/code-du-patrimoine/partie-legislative/livre-ier-dispositions-communes-a-lensemble-du-patrimoine-culturel/titre-ii-acquisition-de-biens-culturels/chapitre-3-preemption-des-uvres-dart/l123-1.md'>L. 123-1</a>, <a href='/code-du-patrimoine/partie-legislative/livre-ier-dispositions-communes-a-lensemble-du-patrimoine-culturel/titre-ii-acquisition-de-biens-culturels/chapitre-3-preemption-des-uvres-dart/l123-2.md'>L. 123-2 </a>et <a href='/code-du-patrimoine/partie-legislative/livre-ier-dispositions-communes-a-lensemble-du-patrimoine-culturel/titre-ii-acquisition-de-biens-culturels/chapitre-3-preemption-des-uvres-dart/l123-4.md'>L. 123-4 </a>sont applicables en Polynésie française.</p><p>Pour l'application de ces articles, les références au code de commerce sont remplacées par les références aux dispositions applicables localement ayant le même objet.</p><p>III.-Les articles <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ii-archives-audiovisuelles-de-la-justice/chapitre-1er-constitution/l221-1.md'>L. 221-1 </a>à <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ii-archives-audiovisuelles-de-la-justice/chapitre-1er-constitution/l221-5.md'>L. 221-5</a>, <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ii-archives-audiovisuelles-de-la-justice/chapitre-2-communication-et-reproduction/l222-1.md'>L. 222-1 </a>et <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ii-archives-audiovisuelles-de-la-justice/chapitre-2-communication-et-reproduction/l222-3.md'>L. 222-3 </a>sont applicables en Polynésie française, dans leur rédaction résultant de la loi n° 2019-222 du 23 mars 2019 de programmation 2018-2022 et de réforme pour la justice.</p>
