# Article L730-3

<p>Pour son application à Mayotte, dans le c de l'article <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ier-regime-general-des-archives/chapitre-1er-dispositions-generales/l211-4.md'>L. 211-4 </a>et dans le d du 4° du I de l'article <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ier-regime-general-des-archives/chapitre-3-regime-de-communication/l213-2.md'>L. 213-2</a>, après les mots : " officiers publics ou ministériels ", sont insérés les mots : " et des cadis ". Dans la deuxième phrase du I de l'article <a href='/code-du-patrimoine/partie-legislative/livre-ii-archives/titre-ier-regime-general-des-archives/chapitre-3-regime-de-communication/l213-3.md'>L. 213-3</a>, après le mot : " notaires ", il est procédé à la même insertion.</p>
