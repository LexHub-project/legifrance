# Article L730-4

<p>Pour l'application du code dans la collectivité départementale de Mayotte, les termes énumérés ci-après sont remplacés comme suit :</p><p>a) (Abrogé) ;</p><p>b) Les mots : " cour d'appel " par les mots : " chambre d'appel de Mamoudzou " ;</p><p>c) Le mot : " département " par le mot : " collectivité départementale de Mayotte " ;</p><p>d) Le mot : " préfet " par les mots : " préfet de Mayotte ".</p><br/><br/><i>NOTA:<p>Conformément à l’article 36 de l’ordonnance n° 2019-964 du 18 septembre 2019, les présentes dispositions entrent en vigueur le 1<sup>er</sup> janvier 2020.</p></i>
