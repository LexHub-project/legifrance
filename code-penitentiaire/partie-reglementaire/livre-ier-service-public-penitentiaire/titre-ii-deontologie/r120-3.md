# Article R120-3

<p><br/>Le code de déontologie du service public pénitentiaire est remis individuellement par l'administration pénitentiaire à chacune des personnes auxquelles il s'applique. Il est affiché dans les établissements et services pénitentiaires. Cet affichage est réalisé de telle façon que le code de déontologie soit également porté à la connaissance des personnes placées sous main de justice.</p>
