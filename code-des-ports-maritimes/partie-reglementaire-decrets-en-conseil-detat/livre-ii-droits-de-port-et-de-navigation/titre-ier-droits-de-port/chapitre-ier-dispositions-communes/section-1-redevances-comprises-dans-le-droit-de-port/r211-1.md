# Article R*211-1

Le droit de port est dû à raison des opérations commerciales ou des séjours des navires effectués dans le port. Les éléments constitutifs du droit de port comprennent, dans les conditions définies au présent code, les redevances suivantes :   1° Pour les navires de commerce :   a) Une redevance sur le navire ;   b) Une redevance de stationnement ;   c) Une redevance sur les marchandises ;   d) Une redevance sur les passagers ;   2° Pour les navires de pêche, une redevance d'équipement des ports de pêche ;   3° Pour les navires de plaisance ou de sport, une redevance d'équipement des ports de plaisance.
