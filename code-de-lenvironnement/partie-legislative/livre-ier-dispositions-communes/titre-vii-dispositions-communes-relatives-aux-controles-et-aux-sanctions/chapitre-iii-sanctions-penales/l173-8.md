# Article L173-8

Les personnes morales reconnues pénalement responsables dans les conditions prévues à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>l'article 121-2 </a>du code pénal des infractions délictuelles prévues au présent code encourent, outre l'amende dans les conditions fixées à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>l'article 131-38 </a>du code pénal, les peines prévues aux 3°, 4°, 5°, 6°, 8° et 9° de <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>l'article 131-39</a> du même code ainsi que celle prévue au 2° de ce même article, qui, si elle est prononcée, s'applique à l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.
