# Article R723-3

<p><br/>Les activités opérationnelles sont exercées par les sapeurs-pompiers volontaires ayant atteint le grade minimum :<br/>1° De sapeur, pour les activités d'équipier ;<br/>2° De caporal, pour les activités de chef d'équipe ;<br/>3° De sergent, pour les activités de chef d'agrès d'un engin comportant une équipe ;<br/>4° D'adjudant, pour les activités de chef d'agrès tout engin ;<br/>5° De lieutenant, pour les activités de chef de groupe ;<br/>6° De capitaine, pour les activités de chef de colonne ;<br/>7° De commandant, pour les activités de chef de site.</p>
