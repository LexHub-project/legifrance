# Article L053-5

L'employeur prend toutes dispositions nécessaires en vue de prévenir les faits de harcèlement sexuel, d'y mettre un terme et de les sanctionner. <p>Le texte de <a href='/code-penal/partie-legislative/livre-ii-des-crimes-et-delits-contre-les-personnes/titre-ii-des-atteintes-a-la-personne-humaine/chapitre-ii-des-atteintes-a-lintegrite-physique-ou-psychique-de-la-personne/section-3-des-agressions-sexuelles/paragraphe-4-de-lexhibition-sexuelle-et-du-harcelement-sexuel/222-33.md'>l'article 222-33</a> du code pénal est affiché dans les lieux de travail ainsi que dans les locaux ou à la porte des locaux où se fait l'embauche.</p>
