# Article L480-4-2

Les personnes morales peuvent être déclarées responsables dans les conditions prévues par l'article 121-2 du code pénal des infractions définies aux articles <a href='/code-de-lurbanisme/partie-legislative/livre-ier-regles-generales-damenagement-et-durbanisme/titre-vi-sanctions-et-servitudes/l160-1.md' title='Code de l'urbanisme - art. L160-1 (V)'>L. 160-1</a>, <a href='/code-de-lurbanisme/partie-legislative/livre-iv-regime-applicable-aux-constructions-amenagements-et-demolitions/titre-viii-infractions/l480-3.md' title='Code de l'urbanisme - art. L480-3 (V)'>L. 480-3</a>, <a href='/code-de-lurbanisme/partie-legislative/livre-iv-regime-applicable-aux-constructions-amenagements-et-demolitions/titre-viii-infractions/l480-4.md' title='Code de l'urbanisme - art. L480-4 (V)'>L. 480-4</a>, <a href='/code-de-lurbanisme/partie-legislative/livre-iv-regime-applicable-aux-constructions-amenagements-et-demolitions/titre-viii-infractions/l480-4-1.md' title='Code de l'urbanisme - art. L480-4-1 (V)'>L. 480-4-1</a>, <a href='/code-de-lurbanisme/partie-legislative/livre-iv-regime-applicable-aux-constructions-amenagements-et-demolitions/titre-viii-infractions/l480-12.md' title='Code de l'urbanisme - art. L480-12 (V)'>L. 480-12</a> et <a href='/code-de-lurbanisme/partie-legislative/livre-v-implantation-des-services-etablissements-et-entreprises/titre-ier-dispositions-administratives-generales/l510-2.md' title='Code de l'urbanisme - art. L510-2 (V)'>L. 510-2 </a>du présent code. Les peines encourues par les personnes morales sont les suivantes : 1° L'amende, suivant les modalités prévues par l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md' title='Code pénal - art. 131-38 (V)'>131-38 du code pénal </a>; 2° Les peines mentionnées aux 2°, 3°, 4°, 5° et 9° de l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md' title='Code pénal - art. 131-39 (V)'>131-39 du code pénal</a>. L'interdiction mentionnée au 2° de l'article 131-39 du même code porte sur l'activité dans l'exercice ou à l'occasion de laquelle l'infraction a été commise.<br/><br/><i>NOTA:</i>
