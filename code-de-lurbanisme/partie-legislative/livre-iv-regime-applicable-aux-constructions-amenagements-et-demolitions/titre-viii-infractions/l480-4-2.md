# Article L480-4-2

<p>Les personnes morales peuvent être déclarées responsables dans les conditions prévues par <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>l'article 121-2 </a>du code pénal des infractions définies aux <a href='/code-de-lurbanisme/partie-legislative/livre-iv-regime-applicable-aux-constructions-amenagements-et-demolitions/titre-viii-infractions/l480-3.md' title='Code de l'urbanisme - art. L480-3 (V)'>articles L. 480-3, L. 480-4, L. 480-4-1, </a><a href='/code-de-lurbanisme/partie-legislative/livre-iv-regime-applicable-aux-constructions-amenagements-et-demolitions/titre-viii-infractions/l480-12.md' title='Code de l'urbanisme - art. L480-12 (V)'>L. 480-12</a>, <a href='/code-de-lurbanisme/partie-legislative/livre-v-implantation-des-services-etablissements-et-entreprises/titre-ier-dispositions-administratives-generales/l510-2.md' title='Code de l'urbanisme - art. L510-2 (V)'>L. 510-2 </a>et <a href='/affichCodeArticle.do?cidTexte=LEGITEXT000006074075&idArticle=LEGIARTI000031213636&dateTexte=&categorieLien=cid' title='Code de l'urbanisme - art. L610-1 (VD)'>L. 610-1</a> du présent code. </p><p>Les peines encourues par les personnes morales sont les suivantes : </p><p>1° L'amende, suivant les modalités prévues par l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>131-38 du code pénal </a>; </p><p>2° Les peines mentionnées aux 2°, 3°, 4°, 5° et 9° de l'article <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>131-39 du code pénal</a>. </p><p>L'interdiction mentionnée au 2° de l'article 131-39 du même code porte sur l'activité dans l'exercice ou à l'occasion de laquelle l'infraction a été commise.</p><br/><br/><i>NOTA:</i>
