# Article R214-3

<p>Le droit de préemption institué en application de l'article <a href='/code-de-lurbanisme/partie-legislative/livre-ii-preemption-et-reserves-foncieres/titre-ier-droits-de-preemption/chapitre-iv-droit-de-preemption-des-communes-sur-les-fonds-artisanaux-les-fonds-de-commerce-et-les-baux-commerciaux/l214-1.md'>L. 214-1 </a>peut s'exercer sur les fonds artisanaux, les fonds de commerce ou les baux commerciaux lorsqu'ils sont aliénés à titre onéreux, à l'exception de ceux qui sont compris dans la cession d'une ou de plusieurs activités prévue à l'article <a href='/code-de-commerce/partie-legislative/livre-vi-des-difficultes-des-entreprises/titre-ii-de-la-sauvegarde/chapitre-vi-du-plan-de-sauvegarde/l626-1.md'>L. 626-1 </a>du code de commerce ou dans le plan de cession arrêté en application de l'article <a href='/code-de-commerce/partie-legislative/livre-vi-des-difficultes-des-entreprises/titre-iii-du-redressement-judiciaire/chapitre-ier-de-louverture-et-du-deroulement-du-redressement-judiciaire/l631-22.md'>L. 631-22 </a>ou des articles <a href='/code-de-commerce/partie-legislative/livre-vi-des-difficultes-des-entreprises/titre-iv-de-la-liquidation-judiciaire/chapitre-ii-de-la-realisation-de-lactif/section-1-de-la-cession-de-lentreprise/l642-1.md'>L. 642-1</a> à L. 642-17 du code de commerce.</p>
