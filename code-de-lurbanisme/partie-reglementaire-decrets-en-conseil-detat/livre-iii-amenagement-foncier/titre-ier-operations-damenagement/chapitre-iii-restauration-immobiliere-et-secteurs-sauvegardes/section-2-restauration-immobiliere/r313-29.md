# Article R313-29

<p>Lorsque l'opération est située dans le périmètre d'un site patrimonial remarquable classé en application de l'article <a href='/code-du-patrimoine/partie-legislative/livre-vi-monuments-historiques-sites-et-espaces-proteges/titre-iii-sites/chapitre-ier-classement-au-titre-des-sites-patrimoniaux-remarquables/l631-1.md' title='Code du patrimoine - art. L631-1 (V)'>L. 631-1</a> du code du patrimoine, l'architecte des bâtiments de France accompagne, s'il y a lieu, pour l'application du <a href='/code-general-des-impots-cgi/livre-premier-assiette-et-liquidation-de-limpot/premiere-partie-impots-detat/titre-premier-impots-directs-et-taxes-assimilees/chapitre-premier-impot-sur-le-revenu/section-ii-revenus-imposables/2e-sous-section-revenu-global/i-revenu-imposable/156.md'>3° du I de l'article 156 du code général des impôts </a>ou de l'article <a href='/code-general-des-impots-cgi/livre-premier-assiette-et-liquidation-de-limpot/premiere-partie-impots-detat/titre-premier-impots-directs-et-taxes-assimilees/chapitre-premier-impot-sur-le-revenu/section-v-calcul-de-limpot/ii-impot-sur-le-revenu/19-septies-reduction-dimpot-accordee-au-titre-des-depenses-de-restauration-immobiliere-dans-les-secteurs-sauvegardes-les-quartiers-anciens-degrades-et-les-zones-protegees/199-tervicies.md'>199 tervicies </a>du même code, son accord sur les travaux projetés d'une attestation certifiant que ces travaux constituent la restauration complète de l'immeuble concerné.</p>
