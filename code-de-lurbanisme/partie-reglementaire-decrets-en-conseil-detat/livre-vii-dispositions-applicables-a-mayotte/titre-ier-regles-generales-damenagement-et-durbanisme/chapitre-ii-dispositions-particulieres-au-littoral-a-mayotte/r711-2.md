# Article R*711-2

<p>Pour l'application de <a href='/code-de-lurbanisme/partie-reglementaire-decrets-en-conseil-detat/livre-ier-regles-generales-damenagement-et-durbanisme/titre-iv-dispositions-speciales-a-certaines-parties-du-territoire/chapitre-vi-dispositions-particulieres-au-littoral/r146-2.md'>l'article R*46-2</a>, les mots : " dans les cas prévus par les articles <a href='/code-de-lenvironnement/partie-reglementaire/livre-ier-dispositions-communes/titre-ii-information-et-participation-des-citoyens/chapitre-iii-enquetes-publiques-relatives-aux-operations-susceptibles-daffecter-lenvironnement/section-1-champ-dapplication-et-objet-de-lenquete-publique/r123-1.md'>R. 123-1 à R. 123-33 du code de l'environnement</a> ” sont remplacés par les mots : " dans les cas où le représentant de l'Etat l'a prévu ".</p>
