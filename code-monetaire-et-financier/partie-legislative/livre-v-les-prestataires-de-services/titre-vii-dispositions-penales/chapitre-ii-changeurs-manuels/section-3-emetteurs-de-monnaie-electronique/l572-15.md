# Article L572-15

<p align='left'>Les personnes morales déclarées pénalement responsables, dans les conditions prévues à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-ii-de-la-responsabilite-penale/chapitre-ier-dispositions-generales/121-2.md'>l'article 121-2 </a>du code pénal, de l'infraction prévue à <a href='/code-monetaire-et-financier/partie-legislative/livre-v-les-prestataires-de-services/titre-vii-dispositions-penales/chapitre-ii-changeurs-manuels/section-3-emetteurs-de-monnaie-electronique/l572-13.md'>l'article L. 572-13 </a>du présent code encourent :</p><p>1° L'amende, suivant les modalités prévues à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-38.md'>l'article 131-38 </a>du code pénal ;</p><p>2° Les peines complémentaires mentionnées à <a href='/code-penal/partie-legislative/livre-ier-dispositions-generales/titre-iii-des-peines/chapitre-ier-de-la-nature-des-peines/section-2-des-peines-applicables-aux-personnes-morales/sous-section-1-des-peines-criminelles-et-correctionnelles/131-39.md'>l'article 131-39</a> du même code.</p><p>L'interdiction mentionnée au 2° de ce même article 131-39 porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle l'infraction a été commise.</p>
