# Article L214-1

I. - Les organismes de placements collectifs sont :   1. Les organismes de placement collectif en valeurs mobilières ;   2. Les fonds communs de créance ;   3. Les sociétés civiles de placement immobilier.   4. Les sociétés d'épargne forestière. L'arrêté d'homologation du 15 mai 2007 a été publié au Journal officiel du 16 mai 2007.</i>
