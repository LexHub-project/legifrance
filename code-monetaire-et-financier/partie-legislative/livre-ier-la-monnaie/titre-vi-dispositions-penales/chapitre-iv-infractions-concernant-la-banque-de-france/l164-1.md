# Article L164-1

Est puni des peines prévues à l'article <a href='/code-penal/partie-legislative/livre-ii-des-crimes-et-delits-contre-les-personnes/titre-ii-des-atteintes-a-la-personne-humaine/chapitre-vi-des-atteintes-a-la-personnalite/section-4-de-latteinte-au-secret/paragraphe-1-de-latteinte-au-secret-professionnel/226-13.md'>226-13 </a>du code pénal, le fait, pour les membres du conseil général, de violer le secret professionnel institué à l'article <a href='/code-monetaire-et-financier/partie-legislative/livre-ier-la-monnaie/titre-iv-la-banque-de-france/chapitre-ii-organisation-de-la-banque/section-2-le-conseil-general/l142-3.md'>L. 142-3</a>, sous réserve des dérogations prévues à l'article <a href='/code-penal/partie-legislative/livre-ii-des-crimes-et-delits-contre-les-personnes/titre-ii-des-atteintes-a-la-personne-humaine/chapitre-vi-des-atteintes-a-la-personnalite/section-4-de-latteinte-au-secret/paragraphe-1-de-latteinte-au-secret-professionnel/226-14.md'>226-14</a> du code pénal.
