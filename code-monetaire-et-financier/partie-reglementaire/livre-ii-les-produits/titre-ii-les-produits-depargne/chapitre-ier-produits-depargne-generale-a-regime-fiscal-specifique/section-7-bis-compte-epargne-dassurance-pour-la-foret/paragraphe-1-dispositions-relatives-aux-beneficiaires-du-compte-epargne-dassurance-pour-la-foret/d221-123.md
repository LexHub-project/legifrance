# Article D221-123

Conformément aux dispositions de <a href='/code-forestier-nouveau/partie-legislative/livre-iii-bois-et-forets-des-particuliers/titre-v-dispositions-relatives-a-lassurance/chapitre-ii-compte-epargne-dassurance-pour-la-foret/l352-4.md'>l'article L. 352-4 </a>du code forestier, les dépôts sur le compte sont autorisés pendant dix ans après la date du retrait effectué dans les conditions mentionnées à <a href='/code-forestier-nouveau/partie-legislative/livre-iii-bois-et-forets-des-particuliers/titre-v-dispositions-relatives-a-lassurance/chapitre-ii-compte-epargne-dassurance-pour-la-foret/l352-3.md'>l'article L. 352-3</a> de ce même code dans la limite du montant du capital inscrit sur le compte le jour précédant le retrait.<br/><br/>
