# Article R532-31

Pour l'application de <a href='/code-monetaire-et-financier/partie-legislative/livre-v-les-prestataires-de-services/titre-iii-les-prestataires-de-services-dinvestissement/chapitre-ii-conditions-dexercice-de-la-profession/section-3-regles-specifiques-concernant-les-pays-tiers/sous-section-1-dispositions-generales/l532-28.md' title='Code monétaire et financier - art. L532-28 (V)'>l'article L. 532-28</a>, l'expression : " Etat membre de référence d'un gestionnaire établi dans un pays tiers ” désigne : <br/><br/>1° Lorsque le gestionnaire a l'intention de gérer un ou plusieurs FIA de l'Union européenne établis dans un même Etat membre, sans le ou les commercialiser dans l'Union européenne : l'Etat membre d'origine de ce ou ces FIA ; <br/><br/>2° Lorsque le gestionnaire a l'intention de gérer plusieurs FIA de l'Union européenne établis dans différents Etats membres sans les commercialiser dans l'Union européenne : l'Etat membre dans lequel la majorité des FIA sont établis ou celui dans lequel le plus grand volume d'actifs est géré ; <br/><br/>3° Lorsque le gestionnaire a l'intention de commercialiser un FIA de l'Union européenne dans un Etat membre : <br/><br/>― l'Etat membre dans lequel le gestionnaire a l'intention de commercialiser le FIA s'il n'a pas été agréé ou enregistré dans l'Union européenne ; ou <br/><br/>― l'Etat membre d'origine du FIA ; ou <br/><br/>― l'Etat membre dans lequel le gestionnaire a l'intention de commercialiser le FIA s'il a été agréé ou enregistré dans un Etat membre ; <br/><br/>4° Lorsque le gestionnaire a l'intention de commercialiser un FIA de pays tiers dans un seul Etat membre : cet Etat membre ; <br/><br/>5° Lorsque le gestionnaire a l'intention de commercialiser un FIA de l'Union européenne dans différents Etats membres : <br/><br/>― l'Etat membre d'origine du FIA ou l'un des Etats membres dans lequel le gestionnaire a l'intention de commercialiser effectivement le FIA si le FIA est agréé ou enregistré dans un Etat membre ; ou <br/><br/>― l'un des Etats membres dans lequel le gestionnaire a l'intention de développer une commercialisation effective si le FIA n'est pas agréé ou enregistré dans un Etat membre ; <br/><br/>6° Lorsque le gestionnaire a l'intention de commercialiser un FIA de pays tiers dans différents Etats membres : l'un de ces Etats membres ; <br/><br/>7° Lorsque le gestionnaire a l'intention de commercialiser plusieurs FIA de l'Union européenne : <br/><br/>― l'Etat d'origine de ces FIA ou l'Etat dans lequel le gestionnaire a l'intention de développer la commercialisation effective de la plupart de ces FIA s'ils sont agréés ou enregistrés dans le même Etat membre ; ou <br/><br/>― l'Etat dans lequel le gestionnaire a l'intention de développer la commercialisation effective de la plupart de ces FIA s'ils ne sont pas agréés ou enregistrés dans le même Etat membre ; <br/><br/>8° Lorsque le gestionnaire a l'intention de commercialiser plusieurs FIA de l'Union européenne et de pays tiers ou plusieurs FIA de pays tiers dans l'Union européenne : l'Etat membre dans lequel il a l'intention de développer la commercialisation effective de la plupart de ces FIA.
