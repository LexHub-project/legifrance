# Article D549-4

<p>Les informations rendues publiques par un dispositif de publication agréé conformément à l'article <a href='/code-monetaire-et-financier/partie-legislative/livre-v-les-prestataires-de-services/titre-iv-autres-prestataires-de-services/chapitre-ix-les-prestataires-de-services-de-communication-de-donnees/section-3-dispositifs-de-publication-agrees/l549-11.md' title='Code monétaire et financier - art. L549-11 (VD)'>L. 549-11</a> comprennent au moins les éléments suivants : </p><p>1° L'identifiant d'identification de l'instrument financier ; </p><p>2° Le prix auquel la transaction a été conclue ; </p><p>3° Le volume de la transaction ; </p><p>4° L'heure de la transaction ; </p><p>5° L'heure à laquelle la transaction a été publiée ; </p><p>6° L'unité de prix de la transaction ; </p><p>7° Le code de la plate-forme de négociation sur laquelle la transaction a été exécutée ou le code “ SI ” lorsque la transaction a été exécutée via un internalisateur systématique ou sinon le code “ OTC ” ; </p><p>8° Le cas échéant, un indicateur signalant que la transaction a été soumise à des conditions particulières.</p>
